# 次チャット引き継ぎ 2026-05-26

## 最初に読むこと

このプロジェクトは、AIでTRPGを全部自動化するゲームではなく、AI NPCとの関係性を資産化する「シナリオ攻略型・記憶継承TRPG Webゲーム」。

プレイヤーは探索者を自作し、シナリオごとにNPCとの関係、連絡先、後日談、報酬、記憶を獲得していく。今回の会話では、借り物の無料シナリオではなく、オリジナル短編シナリオをMVP級の一本として育てる方向に寄った。

## 直近の作業結果

オリジナルシナリオ案の骨格を整理した。方向性は「現代日本 / 探索ホラー / NPC信頼攻略 / 平行世界 / バス事故 / 教団儀式」。

仮題候補:

- `きみだけが覚えている事故`
- `死んだきみの友達`
- `空き家に帰る子`
- `あなたの世界で、きみは死んだ`

核になる導入:

- 探索者は何らかのきっかけで平行世界へ迷い込む。
- その世界では、探索者の平行世界側の自分はすでに死亡している。
- 探索者は、死んだ自分の親友だったNPCと出会う。
- NPCは探索者を見て、死んだ親友が生き返ったと思い込み、泣きながら抱きつく。
- 探索者からするとNPCは知らない相手なので、強い戸惑いから始まる。

## 決定済みの重要設定

NPCの未練は次で確定:

> 親友だけが自分を助けて死んだと思っていた。

事故設定:

- 探索者の平行世界側の家族、NPCの両親、平行世界側の探索者が同じバス事故で死亡。
- NPCだけが生き残った。
- NPCはその後、親戚宅に引き取られたが、安全ではない環境から逃げて家出している。
- 性的暴力や虐待の要素は、直接描写せず、非描写・暗示・保護対象として扱う方針。

教団設定は今回の会話で導入決定:

- 悲劇を経験し、この世界を離れて別の世界へ行きたいと願う人たちの集団が存在する。
- 表向きは喪失や事故の当事者が集まる自助グループ。
- 裏では、世界の境界を越える儀式を行う教団のような存在。
- 彼らの儀式が不完全に成立した結果、バス事故と平行世界転移が発生した。
- 教団は完全な悪ではなく、救済を求めた人たちが他者を巻き込んだ存在として扱う。

教団名候補:

- `雨待ちの会`
- `終点のない路線`
- `隣界乗車会`
- `彼岸停留所の会`
- `次の世界へ行く会`

## 真相の方向性

事故の日、教団は旧道の廃バス停で「この世界から降りる儀式」を行っていた。

儀式の目的:

- 教団員たちを「悲劇の起きなかった別世界」へ移す。
- 失った人や壊れた人生を、別の可能性で取り戻す。

失敗理由の候補:

- 儀式に必要な遺物または異物が不完全だった。
- 願いが強すぎて、周囲の世界境界まで巻き込んだ。
- 儀式に必要な「席数」が足りなかった。
- 近くを通ったレジャー行きのバスが、儀式の経路に巻き込まれた。

結果:

- バスは本来のルートにない停留所に停まった。
- 運行記録やGPSに数十秒の空白が残った。
- 「乗っていないはずの誰か」または「降りたはずの誰か」の痕跡が残った。
- NPCは事故の瞬間、平行世界側の探索者に押し出される形で助かった。
- NPCは「親友だけが自分を助けて死んだ」と思い込み、それが未練になった。
- 死んだ探索者の持ち物が世界境界に引っかかり、現在の探索者を呼ぶ錨になった。

## 未確定の重要要素

次に丁寧に詰めるべきもの:

- 教団が使う異物 / 遺物 / アーティファクト。
- それがクトゥルフ神話風なのか、シノビガミのプライズ風なのか、独自怪異なのか。
- 魔術書や儀式書の名前。
- 教団導師の動機。
- 教団員が戦闘相手になる場合の扱い。
- 境界に残った怪異の正体。

現時点の仮案:

- `雨待ちの半券`: 雨の日に存在しない停留所を呼ぶ半券。
- `隣界停留所考`: 境界路線や別世界への乗り換えについて書かれた怪文書または魔術書。
- `境界路線図`: 廃線や事故現場、行方不明事件がつながって見える異常な路線図。

## 帰還条件

元の世界へ戻るには、以下を揃える方向。

- 事故の日と同じ雨。
- 事故発生時刻。
- 存在しない停留所が現れる旧道。
- 教団儀式に使われた異物または遺物。
- 死んだ平行世界側の探索者の持ち物。
- 探索者自身が「死んだ親友の代わりではない」と宣言すること。

死んだ探索者の持ち物候補:

- 交換ノート。
- 壊れた携帯。
- 青いキーホルダー。
- バスの半券。
- 家族写真。

## NPCを連れて帰る条件

NPCを連れて帰るには、ただ手を引くだけでは足りない。

必要条件:

- NPCが探索者を「死んだ親友」ではなく、別世界から来た別人として受け入れる。
- NPCが未練を言葉にする。
- NPC自身が「行く」と選ぶ。
- 両親の墓、親友の部屋、事故現場のうち複数を巡る。
- 探索者が、連れて帰った後の居場所を引き受ける。

連れて帰った場合:

- NPCには戸籍も居場所もない。
- 探索者の家に居候する。
- 以後、強力な相棒になる。
- 同行サポートも遠隔サポートも可能。
- 境界現象や教団残党に狙われる火種も持ち帰る。

相棒性能案:

- `同行サポート`: 探索判定に補正。
- `記憶照合`: 平行世界や境界系の情報を見抜く。
- `危機察知`: 戦闘開始前に一度だけ奇襲を無効化。
- `遠隔サポート`: 同行していない時もスマホ経由で調査補助。
- `境界感覚`: 存在しない駅、停留所、部屋、扉に反応する。

## 戦闘要素の方向性

クトゥルフ神話TRPGやシノビガミのように、ダイスロールによる戦闘や対立を入れる方針。

戦闘を入れやすい場面:

- 空き家に教団の回収係が来る。
- 事故現場で教団員と遭遇する。
- 廃バス停で再儀式を止める。
- 境界内の最終バスで、空席や怪異が探索者/NPCの存在を奪おうとする。

勝利条件は敵HPを0にするだけではなく、次のような複合条件にする。

- 敵を撃退する。
- 儀式進行度を下げる。
- NPCのパニック値を抑える。
- 教団員を説得して離脱させる。
- プライズに相当する異物を奪取する。

## 報酬設計

物語クリア報酬は用意する方針。

確定クリア報酬案:

- 探索点。
- シナリオ記録。
- 称号。
- 境界の記憶片。
- 異物または遺物の欠片。

途中で手に入るもの:

- 交換ノート。
- 壊れた携帯。
- 青いキーホルダー。
- 事故調査ファイル。
- 保護先の名刺。
- 濡れた家族写真。
- 教団資料。
- 魔術書または儀式書。

エンド別報酬:

- NPCを連れて帰る: 連絡先解放、同行/遠隔サポート解放、相棒化。
- 安全な場所に残す: 世界越しの連絡先、後日談、苦いが誠実な救済。
- 真相完全解明: 称号、境界系判定補正、隠し後日談。
- 失敗帰還: 報酬少なめ。ただし後続シナリオの救済フックを残す。
