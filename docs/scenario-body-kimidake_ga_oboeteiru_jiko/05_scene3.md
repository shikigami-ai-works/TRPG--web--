# 05 シーン3 空き家に帰る子

「少し、休める場所がある」

灯はそう言ってから、すぐに口を閉じた。

言ってしまったことを、後悔しているようだった。駅前から離れ、古い住宅街へ入る。低い塀。細い側溝。冬の午後の薄い光。どれも元の世界で見たことがあるような形をしているのに、表札の一字だけが違う家や、見覚えのある角度で曲がるはずの道が一拍遅れて曲がる場所がある。

灯は道を知っていた。

迷わずに歩く。けれど、急がない。急げない。手の中ではライオンのキーホルダーが小さく鳴っている。歩くたびに一度だけ、金具が爪に触れる。

「あの子の家」

角を曲がる前に、灯が言った。

「……たぶん、誰もいない。事故のあと、ずっと」

そこにあったのは、探索者の家だった。

少なくとも、そう見えた。門の位置。玄関までの短い石畳。郵便受けの高さ。窓の形。元の世界の記憶と完全には重ならない。外壁の色が少し淡く、庭木の背が低い。表札の文字も同じ。それでも、違和感は残る。

帰る場所の形をした、誰も帰ってこない家。

郵便受けにはチラシが何枚も押し込まれていた。雨を吸って、端がふやけている。玄関脇の鉢植えは土だけになっていて、枯れた茎が一本、風でかすかに揺れている。灯は門の前で足を止めた。

「鍵、ここ」

彼女は門柱の裏に手を回す。磁石で貼り付いた小さなキーケースが、かすかな音を立てて外れた。

「前に、教えてもらった。忘れものしたとき、勝手に入っていいって。……今は、勝手に入るしかないけど」

玄関の鍵は、驚くほど聞きなれた普通の音で開いた。

中は冷えていた。

人のいない家の冷たさだった。埃っぽいというほどではない。むしろ、最後に出かけた日のまま、きちんと閉じられている。靴箱の上には小さな置物。傘立てに傘が二本。廊下の奥から、止まった家電の匂いがする。冷蔵庫は空で、コンセントが抜かれている。台所の水道をひねると、しばらく濁った水が出て、それから細く澄んだ音になった。

灯は玄関で靴を脱いだあと、すぐには上がらなかった。

「おじゃまします」

誰もいない家に向かって、小さく言う。

その声は、誰かを呼ぶ声ではなかった。勝手に踏み込むことへの許しを、もう返事のできない家の主に求めている声だった。

リビングには、旅行の前に置いていかれた生活が残っていた。ソファの背に畳みかけのブランケット。テーブルの上に、乾いたペンとメモ帳。壁のカレンダーは事故の月で止まっている。赤い丸がついた日付がある。そこだけ、ペンの圧が少し強い。

誕生日。

そして卒業記念。

灯はそれを見て硬直した。

探索者が何か言うより早く、彼女は首を横に振る。泣くのを止めるためではなく、ここで誰かの名前を呼ばないための動きだった。

「ここ、好きだった」

灯は言う。

「あの子の部屋より、リビングのほうが好きだった。家族の声が、ぜんぶ聞こえるから」

家族の声はもうしない。

代わりに、時計だけが動いている。電池が残っているのか、秒針が壁の上で細く刻む。誰もいない家の中で、その音だけが妙に生きていた。

灯はソファの端に座る。深くは座らない。いつでも立てるように、身体を半分浮かせたまま。探索者が近くに座ると、彼女は一度だけそちらを見る。顔を見て、息を詰める。けれど、すぐに目を逸らさなかった。

「わかってはいるんだ」

灯が言う。

「分かってる。ここは、あの子の家で。きみは、あの子じゃない。……でも、ここにいると、分からなくなる」

その正直さは、頼るための言葉ではなく、間違えないための言葉だった。灯は探索者を死んだ親友の場所へ押し込めようとしているのではない。押し込めてしまいそうな自分を、なんとか止めようとしている。

テーブルの下に、小さな紙袋が落ちていた。中身は空だった。袋の口に、ライオンの小さなシールが貼られている。灯がそれを見つけて、手を伸ばしかける。けれど、触れない。

「あの日、ここから出たんだ」

灯は言う。

「いってきますって。みんなでふつうに。すぐに帰ってくると思って」

ふつう、という言葉が、家の中でいちばん遠いものに聞こえた。

しばらく、二人は何もしない。水を飲む。灯が傷に触れないように、袖を少し直す。探索者は部屋の中を見回す。元の世界の記憶に似たものと、似ていないものが混じっている。写真立ては伏せられている。アルバムは棚の中。洗面所には未開封の歯ブラシが残っている。誰かが帰ってきたときのためではない。誰かがもう帰らないことを、片づけきれなかった跡だった。

外で、車が一台通り過ぎる。

灯の肩が小さく跳ねる。

「ごめん」

「ここなら休めると思ったのに、変だね。休める場所なのに、息がしにくい」

それでも、彼女は立ち上がらなかった。

ソファの端で、ライオンのキーホルダーを膝に置く。手のひらをその上に重ねる。閉じ込めるのではなく、落とさないようにする仕草だった。

「少しだけ」

灯は言う。

「少しだけ、ここにいさせて」

誰もいない家は、返事をしない。

ただ、秒針だけが動いている。止まった日付の下で。帰ってこなかった人たちの家で。探索者と灯は、まだ終わっていない事故の中で、初めて座る場所を見つける。

## このシーンで匂わせること / 匂わせないこと

- 「空き家」は親族宅ではなく、平行世界側で死んだ探索者本人の家である
- 家は探索者の元の家に似ているが、細部が少し違う
- 灯はこの家を知っており、死んだ親友との日常の記憶を持っている
- 家は事故の日から時間が止まったように残っている
- 灯は探索者を「あの子」と混同しそうになる自分を、先に言葉で止めようとしている
- ここは危険から完全に離れた安全地帯ではなく、喪失の形をした一時休憩場所である
- 誕生日と卒業記念の旅行が、この家から始まったことを示す

匂わせない:

- 探索者の年齢・性別・顔立ちの固定
- 死んだ親友の具体的な容姿
- 灯が探索者を完全に本人扱いする描写
- 親族宅や親族加害の具体
- 教団の真の目的
- プレゼントの中身
