# 03 シーン1 死んだきみの顔

目を開けたとき、まず気づいたのは、首のうしろの軽さだった。何かを背負っていたはずの肩が、すこし楽になっている。座席の背もたれに頬がついている。窓の外は薄曇り。冷えた空気が、シャツの襟の内側まで届いてくる。

そこがどこなのかが、すぐには分からなかった。夜行バスの座席にも見えるし、あるいは、見知らぬ駅前のロータリーに停めた誰かの車の助手席だったかもしれない。記憶のうしろ側が、薄い霧で覆われている。覚えているのは、自分こと。家族の顔。それだけ。

外に出る。空気は冬の入り口の匂いをしている。歩道の隅に、雪の溶け残りが薄く広がっている。歩いている人の数も、コンビニの並びも、信号の鳴き声も、いつもの街と同じだった。同じなのに、何かがずれている、という感覚だけが、頭の中を満たしてゆく。

駅前の銀行の看板を見上げる。「しらい銀行」と書いてある。配色も書体も覚えのある通りで、ただ最初の一文字だけが置き換わっている。「みらい」が「しらい」に。立ち止まって何度か読み直す。文字の輪郭は確かに「し」だ。新聞のスタンドが目に入る。今日の日付に、合わない曜日が印字されている。スマホを取り出すと、画面の左上は圏外でも圏内でもない、その中間のような電波のない表示で止まっている。地図アプリは現在地を二三十メートル先に置いて、動かない。

頭の芯のあたりで、静かに警告が鳴りはじめる。

そのとき、視界の端で、誰かの動きが止まった。。

駅前広場の柱のそばに、少女が立っている。15歳くらい。コートの裾を、関節が白くなるほど強く握っている。服のあちこちが汚れて乱れていて、袖口から覗く腕には、まだ新しいあざが幾つか。唇の端が切れている。頬の片側に、強く叩かれた跡が残っている。マフラーは肩から半分ずり落ちたまま。彼女は探索者の顔を見て、息を止めた。

止めて、止めて、それでも止まりきれなくて、小さく震える。

唇が動く。声が出るまでに、かなりの時間がかかった。

「……なんで、生きているの....?」

声は喉の奥でかすれて、最後まで言えない。

「……死んだはずじゃ.....」

その声で、街の音が遠ざかる。バスのブレーキ、子どもの笑い声、踏切の鐘。すべてが遠い出来事になっていく。彼女は身じろぎ。それから、半歩前に出る。出てしまったことに自分で驚いて、また下がろうとする。コートの裾を握る指は離れない。

近づいてみると、彼女の目の中にあるのは、驚喜ではない。混乱と、恐怖と、罪悪感と、それから、薄い、ほんとうに薄い安堵が、同じ場所で重なって動けなくなっている。安堵を感じてしまった自分を、彼女はもう責めている。

「ちがう、ちがうって、分かってる」

呼吸の合間に、彼女はそう繰り返す。

「分かってるのに.....」

語尾の前で、いちいち息が落ちる。一度、視線を地面に逃がす。逃がしてから、また顔を上げる。顔を上げる動作が、まるで誰かに約束をするときのもののように見える。置いていかないでほしい、という願いと、置いていかれる前に自分から離れたい、という願いが、ひとつの顔の上で同じだけ揺れている。

「ごめんなさい、あなたの面影が死んだ親友と重なるの」

彼女が言う。

「あなたは誰....？」

その声で、こちらの世界の輪郭が、ようやくはっきりしてくる。看板の一字。新聞の曜日。地図アプリの誤差。それらは、ただ街のずれではなかった。この街では、探索者と同じ顔のかたちをした誰かが、もうこの世にいないということだった。そしてそのことを、誰よりも知っているのは、目の前で泣いてもいいかと聞いてきた、この少女である。

彼女はまだ、相手の名前を呼ばない。呼んだら何かが決まってしまうことを、身体のほうが先に分かっている。代わりに、彼女はもう一度、コートの裾を強く握り直す。指の関節が、もう一度、白くなる。

街の音が、ゆっくり戻ってくる。

## このシーンで匂わせること / 匂わせないこと

- ここは探索者と同じ魂を持つ誰かが死んだ世界らしい、ということ
- 灯はその死を誰よりも近く知っており、目の前の人物に対して「あの子」と「あの子ではない誰か」の両方を見ている
- 灯はすでに親族宅から逃げ出している。服の乱れ・腕のあざ・唇の傷・頬の打撲跡で、何かから逃げてきたことだけを示す
- 灯の安堵は罪悪感とセットで描く。喜びとして単独に描かない
- 探索者の年齢・性別・顔立ちの具体は地の文では固定しない
- 死んだ親友の顔立ちの具体描写はしない

匂わせない:

- 親族宅で何があったかの具体内容
- 教団「帰還の環」の名称はこのシーンでは出さない
- 探索者の容姿が死んだ親友と完全一致するという明示
- ヨグ・ソトース、境界儀式といった用語
