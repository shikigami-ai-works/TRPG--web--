# 09 シーン7 送り火

中央の炉は、もう教団のものではなかった。

真壁の声は消えている。別室の祈りも遠い。壁の向こうで誰かが泣いているような気配はある。けれど、それはもう命令ではなかった。大勢の願いが炉へ流れ込み、炉がそれを束ね、真壁が道を開く。そういう仕組みは、四方の部屋を乱した時点で壊れている。

残っているのは、形だけだ。

棺型の炉。

四方の留め具。

事故現場から盗まれた遺留品。

まだ空いたままの中心の場所。

そして、灯がまだ開けずに持っている誕生日プレゼント。

灯は炉の前で立ち止まった。

「ここに置いたら」

声が少しだけ震える。

「もう、渡せないんだね」

未開封の包みは、灯の手の中で小さく見えた。血の跡のような茶色い染み。ライオンの形をしたシール。渡せなかった時間が、そのまま紙袋の口を留めている。

中身を知る必要はない。

知ってしまえば、灯の未練は形を持つ。形を持った未練は、もう一度「あの子」のためだけのものになる。ここで必要なのは、プレゼントの中身を確認することではなかった。灯が、それを開けないまま返せるかどうかだった。

灯は探索者を見る。

「開けないでって言ったの、覚えてる」

問いではない。

責めてもいない。

ただ、最後に確かめている。目の前の人が、死んだ親友の代わりとしてそこにいるのか。それとも、灯の未練を勝手に覗かず、灯が灯として選ぶことを待てる人なのか。

探索者が頷くなら、灯は小さく息を吐く。

「ありがとう」

短い言葉だった。

けれど、その短さの中に、ここまで来るまでの道が入っていた。駅前で出会ったときの怯え。空き家での沈黙。帰還の環の記録棚。四方の部屋で交わした言葉。親族夫婦の前で立ち止まった足。真壁が消えたあとの無人の炉。

灯は包みを、炉の縁に置く。

まだ中へは落とさない。

「あの子に渡すはずだった」

言葉は、ゆっくり出た。

「でも、あの子はもういない」

炉の底で、火のない火が揺れる。

「あなたは、あの子じゃない」

灯は探索者を見る。

言い切ったあとで、少しだけ泣きそうな顔になった。残酷なことを言った顔ではない。ようやく、残酷だった事実を自分の口で言えた顔だった。

「似てる。たぶん、すごく。声も、仕草も、変なところで黙るところも。でも、違う」

灯は笑おうとして、うまくいかなかった。

「違う人なのに、ここまで来てくれた」

それで十分だった。

十分だと言うには、あまりにも遅くて、あまりにも血がつきすぎている。けれど、ここで初めて、探索者は死んだ親友の空席ではなく、灯の隣に立つ別の誰かとして見られている。

四方のアーティファクトを、炉の周りで確かめていく。

境界の火種は、前へ戻せそうだ。

空席の名札は、後へ戻せそうだ。

止まった懐中時計は、左へ戻せそうだ。

灰の小瓶は、右へ戻せそうだ。

すべてを戻せば、道は細く安定する。ひとつだけを持ち出すなら、そのぶん炉の輪はわずかに欠ける。持ち帰れるものは、力ではなく痕跡だ。欲張れば、道は重さに耐えられない。

灯は空席の名札を見た。

「これ、持っていったら、覚えていられるのかな」

すぐに首を振る。

「ううん。違うね。持っていくから覚えてるんじゃなくて、覚えてるから、持っていけるんだ」

その理解が、灯の声を少しだけ強くした。

それから灯は、炉の縁に置いた未開封の誕生日プレゼントを見た。

ライオンの形をしたシール。茶色く染みた紙袋の端。渡せなかったまま、事故の日からここまで運ばれてきたもの。

「これを使えば」

灯は、言いかけてから探索者を見る。

「あなたを、向こうへ返せるのかな」

それは、灯が死んだ親友へ渡すはずだったものだった。

宛て先は、平行世界側の探索者。もう死んでしまった、灯の「あの子」。持ち主になるはずだった人がこの世界にいないから、炉はそれを燃料として受け取れる。灯はそう気づいてしまう。自分の手元に残ったものではなく、死者へ届かなかったものなら、探索者を返すために使えるかもしれないと思ってしまう。

けれど、探索者はそれを受け取らない。

燃料にならないからではない。

むしろ、燃料にはなる。だからこそ受け取らない。これは探索者を帰すための切符ではなく、灯が死んだ親友へ渡せなかったものだ。ここで探索者が受け取ってしまえば、灯のさよならまで奪って、もう一度「あの子」の位置に座ることになる。

あるいは、ただ受け取りたくないからでもいい。

これ以上、灯から奪って帰りたくない。灯が自分の手で返すべきものまで、帰還のための材料にしたくない。その拒絶は、儀式の理屈より先に、探索者が灯を「あの子の代わり」ではなく灯自身として見ていることの証になる。

灯はプレゼントを抱え直す。

「……そっか」

短い声だった。

「これは、あの子へ返すものなんだ」

その言葉のあとで、灯は炉の中心を見た。

四方のアーティファクトを戻す場所は、もう分かっている。

未開封の誕生日プレゼントも、盗まれたキーホルダーも、送り火として返せる。

けれど、二人分の帰還儀式にするための媒介だけが、まだ足りない。

「向こうへ」

灯が、小さく言う。

「向こう側へ、届いてほしいもの」

探索者を返すために未開封のプレゼントを差し出そうとしたとき、足りなかったもの。死者の遺留物として燃えるものではなく、境界を越えようとする願い。向こう側にいる誰かへ、手を伸ばし続けた二人分の思い。

灯の顔が、ゆっくりと変わる。

思いついてしまった顔だった。

「あの二人」

声は掠れていた。

「夫婦、だった」

親族夫婦。

灯を縛った人たち。戻れない場所にした人たち。けれど、最後まで二人一組で儀式に加担していた人たち。二人で失い、二人で灯に重ね、二人でここへ来た人たち。

その手にある結婚指輪なら、二人分の媒介になるかもしれない。

それは、倒してすぐに引き抜くものではなかった。

殺めた報酬でも、便利な儀式材料でもない。灯が未開封のプレゼントを差し出そうとして、それを探索者が断って、初めて「燃えるもの」と「境界を越える願い」は同じではないと分かったから思いつけたものだった。

灯は目を伏せる。

「取りに、戻らなきゃ」

その声には、嫌悪がある。

恐怖もある。

でも、逃げだけではなかった。

探索者と灯は、親族夫婦の倒れた場所へ戻る。

廊下は短いはずなのに、長く感じる。さっき通った四方の部屋の前を、もう一度歩く。境界の火種は弱く揺れ、空席の名札は白いままそこにあり、止まった懐中時計は同じ時刻を指し、灰の小瓶は床の端で光を受けている。

倒れた二人の前で、灯は膝をつかない。

近づきすぎもしない。

探索者が手を伸ばすなら、灯は目を逸らさずに見る。灯が手を伸ばすなら、探索者はそこに立つ。どちらにしても、これは灯だけの行為ではない。

結婚指輪は、静かに外される。

祝福のために作られた輪。

誰かと誰かを結びつけるための輪。

それが、ここでは別の形で使われる。

灯は指輪を受け取って、すぐには握り込まなかった。

「ごめんなさいって言えば、済むことじゃない」

彼女は言った。

「でも、持っていく」

それ以上の言葉はない。

中央の炉へ戻る。

炉の中心に、親族夫婦の結婚指輪を置く。

二つの輪は、祝福のために作られたものだった。けれど、祝福だけを運んできたわけではない。失った息子への執着。灯に重ねた影。帰還の環に差し出した願い。二人で抱え、二人で間違え、二人でここまで来た輪だった。

それを、探索者と灯のために使う。

親族夫婦は、境界の向こう側を願っていた二人だった。亡くした息子がそこにいると信じ、その願いを帰還の環の炉へ預けていた。だから二つの指輪には、ただ結婚の証というだけではなく、向こう側へ届いてほしいという二人分の思いが染みついている。

灯のキーホルダーがこちら側に残る思いなら、親族夫婦の指輪は向こう側へ手を伸ばす思いだった。

きれいなことではない。

許されることでもない。

ただ、二人分の媒介として、ここに置く。

灯は指輪から目を逸らさなかった。

「あの人たちを許したわけじゃない」

静かに言う。

「わたしがしたことも、なかったことにはならない」

その言葉を、探索者が受け止める。

灯だけの罪にしないなら、炉の火は細く落ち着く。探索者が軽くしようとしたり、灯だけが悪いのだと置き去りにしたりすれば、火は一瞬だけ大きく揺れる。

ここで必要なのは、正しい言葉ではない。

一緒に背負う姿勢だった。

四方のアーティファクトを、決めておいた場所へ戻していく。

境界の火種は前へ。

空席の名札は後へ。

止まった懐中時計は左へ。

灰の小瓶は右へ。

三つ以上の方角が戻ったとき、炉の音が細く整う。すべてを戻すなら、道はより安定する。ひとつだけを持ち出すなら、その欠けた方角のぶんだけ、灯と探索者は余分な重さを背負うことになる。

炉の前で、境界が開く。

穴ではない。

扉でもない。

元の世界の駅前、夜の道路、雪の残った歩道、コンビニの明かり。そういうものが、水面に映るように炉の底へ揺れている。そこへ足を踏み入れれば、帰れるのかもしれない。あるいは、帰れたと思った瞬間に、別のずれた街へ落ちるのかもしれない。

探索者が儀式の安定を確かめるなら、分かる。

道は細い。

けれど、二人分の重さなら通せる。

四方のアーティファクトを戻しきらなかった場合、または持ち出すものが多すぎる場合、炉は不安定になる。境界は、持ち帰りたいものの数だけ重くなる。灯を連れて帰るなら、物ではなく、関係だけを持ち帰る覚悟が要る。

灯はプレゼントを両手で持ち上げた。

「あの子へ」

それから、言い直す。

「あの子だった人へ」

また、少し首を振る。

「ううん。あの子へ。ちゃんと、さよならを言う」

灯は包みを開けない。

紙袋の口を閉じたまま、炉の中へ差し出す。燃える音はしない。火はない。けれど、包みの輪郭が少しずつ薄くなっていく。焦げるのではなく、雪に溶ける息のように、白くほどけていく。

ライオンのキーホルダーも、炉へ返す。

灯の手元にある片割れではない。事故現場から盗まれ、教団に縛られていた片割れ。灯が抱え続けた未練そのものではなく、誰かが未練として固定したもの。

二つのライオンは、もう揃わない。

だからこそ、戻る道ができる。

灯は自分のキーホルダーを握りしめた。

「これは、持ってる」

誰にも聞かせるためではない声だった。

「忘れないためじゃなくて、忘れても大丈夫なように」

送り火が上がる。

火は赤くない。

白でもない。

灯の顔にだけ、薄い金色の光を落とす。彼女の頬の傷、疲れた目、泣きそうな口元。そのすべてが、誰かの代わりではない少女のものとして照らされる。

炉は、遺物を燃やすのではなく、吐き出すように炎を返してくる。

未開封の包み。盗まれたキーホルダー。二つの結婚指輪。炉へ戻した四方のアーティファクト。それぞれの輪郭が薄くほどけるたび、炉の奥から細い炎が伸びる。炎は上へ昇らず、空席の名札が置かれた椅子のほうへ流れていった。

誰かの代わりに座らせるための椅子。

けれど今は、帰るために座る椅子。

ここで、最後の選択になる。

探索者は、灯と二人で帰ることができる。

そのためには、灯を「あの子」の代わりにしないこと。未開封のプレゼントを開けずに返していること。親族夫婦との罪を灯一人に背負わせていないこと。四方のアーティファクトを欲張って持ち出していないこと。ここまでの会話で、灯が探索者を信じられるだけの距離にいること。

条件が揃っているなら、灯は手を伸ばす。

「一緒に帰っても、いいのかな」

彼女はまだ、帰るという言葉を信じきれていない。

探索者が空席の椅子に座る。

灯は、すぐには近づかない。炉から吐き出される炎を見て、探索者を見て、自分の手の中に残ったライオンのキーホルダーを見る。それから、ほんの少しだけ迷って、探索者の膝の上に腰を下ろす。

探索者が引き寄せるのではない。

灯が、自分でそこを選ぶ。

一つの椅子に、二人分の重さが乗る。同じ椅子の狭さ、膝にかかる重み、肩越しに見える炎。近い距離は、怖さも連れてくる。けれど、灯は逃げない。探索者も、灯を誰かの代わりとして抱えない。ただ、同じ光を見るために、同じ椅子にいる。

炉の炎が、二人の前で明るくなる。

「あったかいね」

灯は小さく言う。

その声が消える前に、光が二人を包む。

灯が自分の足で境界を越える。その隣ではなく、同じ椅子の上で、同じ炎を見ながら。二人は元の世界へ戻る。

探索者だけが帰ることもできる。

灯を置いていくと決めるなら、探索者は空席の椅子に一人で座る。

膝の上には、誰もいない。

炉から吐き出される炎だけが、探索者の顔を照らす。灯は怒らないかもしれない。泣かないかもしれない。ただ、椅子の外に立ち、炉の明かりを見ている。探索者は一人で椅子に座ったまま、その炎と、明かりに照らされた灯の顔を見る。

「大丈夫」

灯はそう言うかもしれない。

「たぶん、今度は、覚えてるから」

その言葉が本当かどうかは分からない。けれど、次にどこかで会うとき、灯の中にこの夜の残響が残る可能性はある。

探索者は、こちらの世界に残ることもできる。

帰らないと選ぶなら、炉の道は探索者の背後で細く閉じていく。灯は驚く。止めようとするかもしれない。自分のために帰れなくなることを、簡単には受け入れない。

「だめだよ」

灯は言う。

「わたしのせいにしないで」

そこで、探索者が灯のせいではないと示せるなら、残る選択は自己犠牲ではなくなる。帰らない罰でも、灯への執着でもない。こちらの世界で、灯と共にこの後を引き受ける選択になる。

どの道を選んでも、送り火は消える。

灯が一緒に帰るなら、炉の光の向こうで、駅前の看板が一字だけ正しい形に戻る。スマホの地図が現在地を思い出す。雪の残った歩道に、同じ椅子から落ちた二人分の足音が重なる。

探索者だけが帰るなら、目を開けた場所に灯はいない。けれど、一人で座った椅子の冷たさ、未開封のまま返されたプレゼントの重さ、炉の明かりに照らされた灯の顔、最後に言われた「大丈夫」の声だけが、遅れて胸に残る。

こちらに残るなら、帰還の環の建物は朝の光の中でただの古い医院に見える。けれど地下には、もう火のない炉がある。灯はその前で、探索者のほうを見ずに言う。

「じゃあ、帰ろう」

元の世界へではない。

この世界の、今日の続きへ。

送り火は、誰かを生き返らせる火ではない。

灯を過去へ戻す火でもない。

死んだ親友を、死んだまま大切にするための火。

探索者を、別の誰かとして受け入れるための火。

そして、灯が自分の名前で次の一歩を踏むための火だった。

## このシーンで扱うこと / 扱わないこと

- 中央の炉は、教団儀式の残骸を二人分の帰還儀式へ組み替える場として扱う
- 未開封の誕生日プレゼントは、所有者になるはずだった平行世界側の探索者が死んでいるため燃料にはなり得るが、探索者を返すためには使わず、開けないまま送り火へ返す
- 盗まれたライオンのキーホルダーは、教団に固定された未練として炉へ返す
- 灯の手元のキーホルダーは、所有者である灯がまだ生きているため燃料にしない。忘れないためではなく忘れても大丈夫になるためのものとして残す
- 親族夫婦の結婚指輪は、境界の向こう側を願っていた二人分の思いを受けた媒介として炉の中心に置く
- 結婚指輪は、未開封の誕生日プレゼントを探索者の帰還燃料にする提案を探索者が断った後で思いつき、親族夫婦の遺体のところへ戻って外す
- True 方向では、灯を親友の代わりではなく別人として受け止めることを明確にする
- Aルートでは、探索者が空席の椅子に座り、灯が自分で探索者の膝の上に座り、二人で炉から吐き出される炎を見る。二人の接触は、所有や依存ではなく、同じ光を見る信頼の証として扱う
- 一人帰還では、探索者だけが空席の椅子に座り、炉の明かりと灯の顔を見ながら光に包まれて元の世界へ戻る
- 最終選択は「灯と二人で帰る」「探索者だけが帰る」「灯と共に残る」の三方向に分ける
- 四方アーティファクトは持ち帰りすぎると境界が不安定になる

扱わない:

- 誕生日プレゼントの中身
- 死んだ親友の復活
- 灯が完全に癒えたという断言
- 親族夫婦や真壁の罪が浄化されたという扱い
- 結婚指輪だけで無条件に帰れるという描写
- 探索者が灯を所有物のように連れ帰る描写
- 残留エンドを自己犠牲や依存だけで美化する描写

## GM向け進行メモ

- `burn_keepsakes_as_farewell` は、未開封の誕生日プレゼントと盗まれたキーホルダーを炉へ返す場面として扱う
- プレゼントを開けていないことは、灯を死んだ親友の代用品にしないための重要条件になる
- `refuse_unopened_gift_as_return_fuel` は、未開封の誕生日プレゼントを探索者の帰還燃料にする案を断る場面として扱う
- 未開封の誕生日プレゼントは、宛て先である平行世界側の探索者が死んでいるため炉の燃料にはなり得る。ただし探索者がそれを受け取ると、灯のさよならを奪い「あの子」の位置に座ることになるため断る
- 灯の手元のキーホルダーは所有者の灯が生きているため、そもそも探索者を返す燃料にはしない
- 親族夫婦の指輪は、境界の向こう側にいると信じた息子を思い続けた二人の願いを受けているため、境界を超える媒介として扱える
- 親族夫婦の指輪は、倒してすぐ引き抜かない。プレゼントを探索者の帰還燃料にする案が否定されてから、燃えるものと境界を越える願いの違いに気づいて思いつく
- `promise_return_together` は、強い約束ではなく「今度は一緒に帰ろう」と言える距離まで来たことの確認にする
- `check_final_boundary_stability` は、儀式の正解探しではなく、持ち帰るものを欲張りすぎていないかを確認する判定にする
- True 方向では `shared_guilt`、`regret_resolved`、`ritual_reproduced`、灯の信頼、持ち帰り数の制限を意識する
- Aルートの膝上描写は、灯が自分で選んで座ることを必ず示す。探索者が一方的に引き寄せる描写にはしない
- 一人帰還は、同じ空席の椅子を使うことでAルートと対比する。膝の上が空いていること、炉の明かりに照らされた灯を見ることを残す
- Normal 方向では、灯を置き去りにするというより、灯がまだ越えられない条件が残った結末として扱う
- Good 方向の残留は、罰や依存ではなく、こちらの世界で続きを引き受ける選択として描く
- Lost 方向では、境界侵食や持ち帰りすぎで道が潰れる。灯や探索者の意思を否定するより、儀式構造が耐えられなかった結果にする
