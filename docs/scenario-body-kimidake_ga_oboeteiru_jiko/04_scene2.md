# 04 シーン2 祝えなかった誕生日

駅前の喧騒から少し離れると、街は急に声を落とした。

線路沿いの歩道。古い街路樹の根元に、溶け残った雪が灰色に固まっている。灯はその横を歩きながら、何度も足を止めかけた。止まらない。けれど、歩幅はさっきより狭くなっている。コートの袖から出た指が、マフラーの端を握ったり、離したりする。

「あの夜も、雪だった」

それだけ言って、灯は口を閉じる。

次の言葉が出てくるまで、信号が一度変わった。車が流れて、止まって、また流れる。灯はその音を聞いているようで、聞いていない。視線はずっと、歩道の端の汚れた雪に落ちている。

「夜行バス。誕生日のそして卒業記念の、旅行だった。わたしと、あの子と、家族で。ちゃんと祝おうって、そういう話で」

誕生日、という単語だけが、他の言葉より少し小さかった。

事故のことを尋ねると、灯は首を横に振る。強くではない。思い出したくないというより、思い出せる形で残っていない、という振り方だった。

「その時をしっかり覚えていないの。白かった。窓の外が、ぜんぶ。誰かがわたしの名前を呼んで、それから、強く抱きしめられた感覚がしたと思ったら。衝撃で気を失った....」

そこで一度、灯は自分の両腕を抱く。抱いたあと、あざに触れたことに気づいたみたいに、すぐ手を離した。

「あの子が、わたしを助けたんだって、だれかが言った。わたしも、そう思ってた。そう思わないと、どうしてわたしだけ残ったのか、分からなかったから」

けど、と言いかけて、灯は言葉を落とす。

けれど、本当にそうだったのか。

それを口にしないまま、彼女はポケットの中から小さなものを取り出した。ライオンのキーホルダーだった。丸くて、少し子どもっぽい顔をしている。金具の輪は何度も開け閉めされた跡があり、耳の塗装が少しだけ剥げている。

「おそろいだった。わたしのは、こっち」

灯はそれを手のひらに乗せる。指で隠すようにして、けれど見せる。大事なものを見せたい気持ちと、見られたくない気持ちが、同じ指の動きに混ざっている。

「あの子のは、なくなった。事故のあと、荷物の中になかった。警察の人も、分からないって」

スマホで事故の記事をたどる。古いニュースサイトの本文は短い。夜行バス。山道。複数名死亡。生存者一名。日付の横に、やはり年月がずれている。関連記事の下に、献花台の写真がある。花束、濡れた道路、白い息。写真の隅に、誰かの手が写っている。警察の制服ではない。遺族でもない。黒いコートの袖口と、薄い紙袋。

灯が画面を見て、息を止める。

「それ....」

指先が震える。彼女は画面には触れない。

「あの子に、渡すはずだった包み。わたしが持ってた。バスの中で渡すつもりで。でも、事故のあと、どうしても見つからなかった」

包みの中身を、灯は言わない。言わないことで、そこに何が入っていたのかよりも、渡せなかったという事実だけが重くなる。開けられなかったもの。言えなかったこと。祝えなかった誕生日。

事故の現場に最初に入った者の記録を追うと、ひとつの名前が浮かぶ。

喪失者支援団体、『帰還の環』。

亡くした人を悼む家族のための会。事故後に献花台を整理し、遺族の相談を受け、現場に残された品を預かったとされる団体。サイトの写真には、穏やかな笑顔の人々と、白い花と、円を描く細いロゴが並んでいる。

灯はその名前を、声に出して読まなかった。

ただ、ライオンのキーホルダーを握る手に、また力が入る。金具が小さく鳴る。街の音に紛れそうなほど小さい音だったのに、探索者にははっきり聞こえた。

「あの子のもの、返してほしい」

灯は言う。

「でも、返ってきたら、わたし、どうすればいいんだろう。」

彼女は探索者の顔を見ない。見たら、さっきと同じように息が止まるからかもしれない。ただ、歩道の雪を見つめたまま、もう一度だけ、キーホルダーを握り直す。

祝えなかった誕生日は、まだ終わっていない。

## このシーンで匂わせること / 匂わせないこと

- 夜行バス事故の日、灯と死んだ親友は誕生日を祝う旅行へ向かっていた
- 灯の事故記憶は、雪、夜、名前を呼ぶ声、冷たい息として断片的に残っている
- 「親友が自分を庇って死んだ」という理解は、灯本人にも確信しきれない
- その曖昧さが、灯の罪悪感と未練になっている
- 灯はいまも自分のライオンのキーホルダーを持っている
- 死んだ親友のライオンのキーホルダーと未開封の誕生日プレゼントは、事故後に不自然に消えている
- 喪失者支援団体「帰還の環」の名前が、遺留品を持ち去った痕跡として浮上する

匂わせない:

- 帰還の環の真の目的
- ヨグ・ソトース、境界儀式、召喚といった用語
- 親族宅で灯に何があったか
- 誕生日プレゼントの中身
- 死んだ親友の顔立ちや、探索者との完全一致
