# 01 シナリオ概要 / 背景真相 / 世界の質感

## シナリオ概要

「きみだけが覚えている事故(仮名)」は、平行世界を題材にしたクトゥルフ神話の流れる短編シナリオである。プレイ時間は約九十分、シーンは七つ。探索者は元いた世界の帰路をどこかで踏み外し、よく似た、しかし一つだけ違う街で目を覚ます。その世界ではその世界の探索者自身はすでに死んでいる、ということ。

街角で、15歳くらいの少女・水瀬灯が探索者に出会う。彼女は驚き。身体を震わせ。喉に詰まった息のまま声をかけてくる。灯にとって探索者の顔は、夜行バス事故で失った親友のものに似ている。年齢が違うから完全には重ならない。それでも、立ち姿、声の細部、視線の癖が、彼女の身体に「あの子」を思い出させる。物語は、ここから始まる。

事故の遺留品は、警察より先に喪失者支援団体「帰還の環」によって持ち去られていた。穏やかな表向きを持つこの団体は、内側で「境界を超える儀式」を進めている。導師・真壁透一郎は、悲しみと罪悪感を餌に集めた信者を燃料にして、まだ名のない通路の向こう側――ヨグ・ソトースの先触れ――を呼ぼうとしている。灯を引き取った親族夫婦も、亡くした息子を取り戻すために、その輪の一部に加担している。多数の構成員は見守り網、記録係、遺留品管理、別室の祈り手として儀式を支えるが、直接戦闘の相手になるのは灯を縛る接続を持つ親族夫婦と、儀式を束ねる真壁である。

探索者と灯は、平行世界側で死んだ探索者本人の空き家へ一度身を寄せる。だが、帰還の環の善意めいた見守り情報網によって居場所は割れ、灯を引き取った親族夫婦が「迎えに来る」。そこで灯は、自分の口で帰らないと選ぶ。その選択を遮らず横に立ったあと、二人は教団本拠地へ向かう。二人は四方の部屋で教団儀式を乱し、それを止めに来た親族夫婦と衝突して、生存のために二人を殺める。この時点では指輪を取得せず、指輪を儀式に使うこともまだ思いつかない。その後、導師・真壁透一郎と戦い、勝敗にかかわらず真壁はいなくなる。無人の儀式場に探索者と灯だけが残されたあと、灯は未開封の誕生日プレゼントを探索者の帰還燃料にしようとする。プレゼントは平行世界側の探索者へ渡されるはずだったもので、宛て先の相手はもう死んでいるため炉の燃料にはなり得る。だが探索者がそれを断って初めて、二人は、ペアである親族夫婦の結婚指輪を二人分の儀式媒介にできるのではないかと思いつく。親族夫婦は境界の向こう側へ息子を求めていた二人だから、その対の指輪が二人分の願いを束ねる媒介になる。二人は親族夫婦の遺体のところへ戻って指輪を取得し、それを使って帰還儀式を再現する。中央の棺型の炉には、事故現場から盗まれた遺留品と、まだ誰にも開かれない誕生日プレゼントが縛られている。それを送り火として返すことで、灯の未練は破壊されるのではなく、彼女の手元から離れていく。

このシナリオの感情的な軸は、ただ一つ。灯を「死んだあの子の代わり」として救わないこと。同じ魂を持つ別の人格として、灯の前に立つこと。True ルートはその一点だけで成立する。

## 背景真相

ある世界に、水瀬灯という少女がいた。彼女には親友がいた。誕生日と卒業記念の旅行に、二人の家族全員で出かけるためにバスに乗った。乗ったのは夜行バス。雪の降る山道。詳しい経緯を、灯は最後まで思い出せない。覚えているのは、自分の名前を呼ぶ声と、強く抱きしめられた感触、白い息と、ガラスの割れる音だけ。記憶はそこまでしかない。

灯は生き残り、親友は死んだ。「あの子だけがわたしを助けて死んだ」と、灯はそう信じている。本当に庇われたのか、ただ運が偏っただけなのかは、彼女自身にも分からない。その曖昧さが、灯の罪になっている。

事故後、彼女を引き取ったのは親族夫婦だった。表向きはまっとうな親戚の善意。だがこの夫婦は、数年前に自分たちの息子を失っていた。喪失を抱え込んだ二人は、いつの間にか「帰還の環」に通うようになっていた。亡くした息子の遺品はすでに教団に差し出され、大会のメダルは棺型の炉の底で燃えかすになっている。灯はそこで、息子の代わりとして暮らすことを求められた。鍵のかかった扉、棘のある優しさ、丁寧すぎる生活音。かりそめの優しさはすぐに剥がれ落ち、灯は親族の家から逃げ出した。何があったのかを、本文は具体的には開かない。服の乱れ、腕のあざ、言葉の詰まりだけが、そこが帰る場所ではなかったことを示している。

帰還の環は、表向きは喪失者の自助グループである。だが導師・真壁透一郎は、元民俗学講師としての知識と、信徒の悲しみそのものを燃料にして、境界の向こう側を観測しようとしている。ヨグ・ソトースの先触れ。まだ境界の道に名はなく、ただ「道が開く」ということだけが、彼の興味の中心にある。信徒には「失った人を取り戻す儀式」と語る。真壁自身も、その言葉を半分は信じている。完全な詐欺師ではない。それがいっそうたちが悪い。大勢の信徒は武装集団ではない。彼らは相談員、見守り係、遺留品管理者、祈り手として輪の外周を作り、真壁がその喪失を炉へ束ねている。

事故現場からは、灯と親友の遺留品の一部が、警察の到着より先に消えていた。おそろいだったライオンのキーホルダーの片割れ。渡されるはずだった誕生日プレゼントの包み。それらは教団施設の中央の炉に縛られ、灯と死んだ親友の未練を境界に固定する燃料として使われている。

そして探索者は、その世界の側から見れば、死んだ親友と「同じ魂」を持つ存在である。同じ魂、別の人生。年齢も、必ずしも一致しない。一致するのは、灯の身体が知っている類いのもの――手の動き、視線の止まり方、言葉の前に置く一拍。真壁はそれを「揺らぎ」と呼び、灯の親族はそれを「儀式を完成させる余剰」と呼ぶ。教団は探索者の出現を、扉が一段開いた証として歓迎している。

帰還儀式を再現するためには、四方の部屋に置かれたアーティファクトを使って教団儀式を一度乱し、親族夫婦との衝突と真壁戦を終える必要がある。親族夫婦はそこで死亡するが、指輪はまだ取得されない。真壁が消え、無人の儀式場に二人だけが残されたあと、灯はいったん未開封の誕生日プレゼントを差し出そうとする。プレゼントは死んだ平行世界側の探索者へ渡されるはずだったものなので燃料にはなり得るが、探索者がそれを断ることで、燃えるものと境界を越える願いは同じではないと分かる。そこで初めて、親族夫婦の結婚指輪が二人分の帰還儀式にふさわしいペアの媒介になるのではないかと思いつく。親族夫婦は境界の向こう側へ息子を求めていた二人であり、その願いが歪められて炉へ束ねられていたから、二つで一組の指輪には二人分の思いを受ける役割が残っている。そこで遺体のところへ戻って指輪を取得し、中央の炉に立つ。罪は、灯一人のものにはできない。共犯であることを認めるか、灯に背負わせて自分だけが軽くなるか。その選択が、後の灯の表情、感情を決める。

最後の場面、棺型の炉の前で、灯は血濡れた未開封の誕生日プレゼントを差し出す。中身を覗かないでほしい、と彼女は言うかもしれない。あの子に渡すはずだったものだから。それを開けずに、ただ送り火として炉に返したとき、灯はようやく、目の前にいるのが「あの子」ではないことを受け入れる。同じ魂であっても、別の人生を生きた、別の誰かであることを。

それがこの世界で起きた事故の、唯一の終わらせ方である。

## 世界の質感(平行世界の手触り)

この世界は、元の世界とほとんど同じである。同じ駅。同じ路線図。同じ薄曇り。空気の匂いも、コンビニの並びも、雪の溶け残った歩道の色も、もとの世界とほぼ変わらない。違いは、目を凝らさなければ見えない場所にある。

駅前の看板の社名が、一字だけ違っている。「みらい銀行」が「しらい銀行」になっている、というふうに。配色も書体もそのまま、ただ一文字だけが、置き換えられている。新聞の日付の曜日が合っていない。月と日は今日のもののはずなのに、印字された曜日が一つずれている。スマホは圏外でもなく、圏内でもない、その中間のような表示で止まっていて、地図アプリが現在地を二三十メートル先に置く。電車のアナウンスは正常で、しかし最後の一語だけが、聞き取れない言葉に置き換わっている。

派手な異界化ではない。風景が歪むことも、空の色が変わることもない。ただ「ほぼ同じなのに、ひとつだけ違う」という違和感が、街のあちこちに微差として置かれている。人々は普通に歩き、普通に話す。誰も平行世界に来てしまったとは気づかない。気づくのは、もう一つの世界を知っている探索者だけである。

その違和感の中心に、水瀬灯が立っている。元の世界ではすれ違ったかもしれない少女が、この世界ではすでに半身を欠いて生きている。雪の山道で誰かを失った身体のまま、こちらの街を歩いている。彼女の隣に立てば、世界の微差が一斉に音を持つ。看板の一字、新聞の曜日、止まりかけの地図アプリ。そのすべてが、ここがあの世界ではないということを、静かに告げてくる。
