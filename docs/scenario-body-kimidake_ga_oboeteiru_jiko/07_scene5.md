# 07 シーン5 帰還の環

帰還の環の本拠地は、思っていたよりも普通の建物だった。

駅から二つ離れた通り。古い医院を改装したような白い外壁。入口のガラス戸には、細い円を重ねたロゴが貼られている。喪失者支援・地域見守り・事故遺族相談。そういう言葉が、柔らかい書体で並んでいた。看板の下には白い鉢植え。よく手入れされている。枯れた葉は一枚もない。

灯はその前で足を止めた。

「ここ」

声は小さい。けれど、初めて見る場所を指す声ではなかった。知っている。少なくとも、名前だけは。親族の家で聞いたことがあるのか。封筒や冊子の端で見たことがあるのか。彼女はガラス戸に映る自分の顔を見て、すぐに目を逸らした。

中へ入ると、空気は暖かかった。

暖かすぎるくらいだった。受付台。消毒液。淡い色のパンフレット。小さなテーブルには個包装の菓子とティーバッグが置かれている。壁には、亡くした人の名前を書き込むためのカード、喪失をひとりで抱えないための言葉、月ごとの集会案内。どれも丁寧で、どれも善意の形をしていた。

けれど、奥へ行くほど、音が少なくなる。

人のいる気配はある。扉の向こうで椅子が引かれる音。紙をめくる音。低い話し声。なのに受付には誰もいない。呼び鈴もない。ここへ来る者は、最初から来ることを知られている。そういう場所の静けさだった。

廊下の壁に、写真が並んでいる。

笑っている人。制服姿の子ども。家族写真。花束を持つ老女。名前の下に、小さな日付。死んだ日付ではない。帰る日、と書かれていた。灯はその文字を読んで、唇を少し開いた。何か言おうとして、言葉にしない。

「帰る日って、なに」

問いではなかった。

答えがほしくて言ったのではなく、言ってしまわなければ目の前の文字に飲み込まれそうだった。

相談室と書かれた部屋の奥に、記録棚があった。鍵はかかっていない。あまりにも自然に開く。ファイルは事故ごと、家族ごと、生還者ごとに分けられていた。夜行バス事故。献花台対応記録。遺留品一時預かり。地域見守り連絡網。水瀬灯の名前。親族夫婦の名前。死んだ親友の家の住所。

空き家の固定電話に残された留守番電話と、同じ書式。

誰が、いつ、どこで灯を見かけたか。どの家に連絡したか。どの事故関係者宅を確認するべきか。すべてが心配という言葉で包まれている。けれど、紙の上でつながれた線は、逃げ道を塞ぐ網にしか見えなかった。

棚の下段には、小さな箱がいくつも並んでいた。

番号札。白い布。透明な袋。ひとつの袋の中に、ライオンのキーホルダーが入っている。灯の手の中にあるものと、よく似た形。こちらは少しだけ角ばっていて、顔つきが強い。金具の輪に、焼けたような黒ずみがついている。

灯の呼吸が止まった。

「あの子の」

指が伸びる。けれど、袋の上で止まる。触っていいのか、触ったら何が終わるのか、分からない手だった。

その隣に、包みがあった。

小さな紙袋。口を留めるシールは、ライオンの形。血の跡のような茶色い染みが端に残っている。灯が渡せなかった誕生日プレゼント。ここまで誰にも開けられず、けれど誰かの手でここまで運ばれてきたもの。

「開けないで」

灯は、探索者が何かする前に言った。

強く言ったわけではない。ただ、早かった。自分でも驚くくらい早く、言葉が出ていた。

「それは、あの子に渡すはずだったものだから」

部屋の奥で、スピーカーが小さく鳴った。

『見つけましたか』

男の声だった。穏やかで、近すぎない。壁のどこかから響いているのに、こちらの息の長さまで測っているような声。

『悲しみは、閉じ込めるものではありません。正しい場所へ返してあげるものです』

灯は袋を胸元に抱えた。ライオンのキーホルダーも、透明な袋ごと手に取る。抱きしめるのではない。落とさないように、奪われないように、ただ持つ。

廊下の向こうで、いくつもの扉が閉まる音がした。

ひとつではない。相談室。集会室。事務室。札のかかった扉の向こうで、椅子が引かれ、紙が揃えられ、誰かが短く咳をする。磨りガラスの奥に、人影がいくつも重なっている。白い袖。黒い服。手帳を抱えた腕。祈るように組まれた指。

けれど、誰もこちらへ出てこない。

逃げるために隠れているのではない。見て見ぬふりをしているのでもない。彼らはここで、すでに自分の役割を持っていた。受付に座る人。記録を整える人。遺留品の袋に番号を貼る人。別室で名前を読み上げる人。帰ってきてほしいと願う人。その一つ一つが、中央の炉へ向かう細い糸になっている。

灯は磨りガラスの向こうを見た。

「たくさん、いる」

声は責めるためのものではなかった。ただ、理解してしまった声だった。ここは空っぽの悪の巣ではない。悲しんでいる人たちがいる。善意の言葉を信じた人たちがいる。けれど、その全員が、灯を助けるためではなく、灯の逃げ道を狭くする輪の一部になっている。

記録棚の横に、今日の配置表が貼られていた。

炉前準備。四方室確認。遺留品返還式。見守り連絡待機。名前の列の中に、灯の親族夫婦の名もあった。空き家の玄関先で聞いた声。その家には戻らないと言ったとき、灯の身体が先に拒んだ人たちの名前。

灯の指が、紙袋の端を握る。

「いるんだ」

小さく言って、彼女はそれ以上を飲み込んだ。

まだ会っていない。けれど、ここにいる。あの家から伸びてきた手は、帰還の環の中で別の形になり、中央の炉のほうへ続いている。

廊下の奥で、またスピーカーが鳴った。

『よく、ここまで辿り着きました』

真壁透一郎の声は、変わらず穏やかだった。

『悲しみは、閉じ込めるものではありません。正しい場所へ返してあげるものです』

扉の向こうで、低い声が重なる。

誰かの名前。帰ってきてほしいという願い。まだ受け入れられないという吐息。言葉ははっきりとは聞こえない。けれど、そのすべてが同じ方向へ流れていることだけは分かった。

『中央の炉へどうぞ。四方の部屋は、もうあなたがたにも開いています』

灯は顔を上げる。

泣いてはいなかった。泣くための場所は、まだここではない。手の中には、透明な袋に入ったライオンのキーホルダー。未開封の誕生日プレゼント。どちらも、ここに置いていけば、誰かの祈りの形に変えられてしまうものだった。

「行くしかないんだね」

問いではなかった。

帰還の環は、人の悲しみを輪にして閉じ込める。

探索者と灯は、その輪の奥へ進む。

## このシーンで匂わせること / 匂わせないこと

- 帰還の環は、表向きの相談施設・見守り窓口・喪失者支援の顔を持っている
- 施設の記録には、事故関係者宅、生還者、遺留品、見守り連絡の情報が整理されている
- 盗まれたライオンのキーホルダーと未開封の誕生日プレゼントは、灯と死んだ親友の未練を縛る遺留品である
- 帰還の環には多数の構成員がいるが、彼らは戦闘員ではなく、見守り網・記録係・遺留品管理・祈り手として儀式を支えている
- 親族夫婦は空き家で終わらず、教団本拠地にいる気配と記録を残す
- 親族夫婦との直接衝突は、このシーンではなく四方の部屋で儀式を乱した後に起こる
- 結婚指輪はこのシーンでは取得しない。儀式に使う発想もまだ出さない
- 真壁は穏やかな声で場を見ており、中央の炉と四方の部屋へ誘導している

匂わせない:

- 誕生日プレゼントの中身
- 親族による加害の具体内容
- 親族夫婦との決着
- 殺害の詳細な身体破壊描写
- 殺害を肯定する文脈
- ヨグ・ソトースの具体名や儀式全体の完全な説明
- 結婚指輪の取得
- 結婚指輪を儀式媒介にする発想
- 結婚指輪で帰還が確定するという断言
