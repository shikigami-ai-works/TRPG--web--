# 08 シーン6 四方の部屋

中央の炉は、地下にあった。

階段を下りるほど、建物の上階にあった相談室の暖かさは遠ざかっていく。白い壁。柔らかいパンフレット。人を安心させるための言葉。その全部が、背中の上で薄い皮のように剥がれていった。

地下の空気は冷たくない。

むしろ、ぬるい。

火があるからではなかった。たくさんの息が、ここへ流れ込んでいるからだ。扉の向こう。壁の奥。別室の椅子に座る人々の祈り。名前を呼ぶ声。帰ってきてほしいという願い。受け入れられないという震え。それらが一本ずつ糸になり、中央の炉へ結ばれている。

灯は、透明な袋に入ったライオンのキーホルダーと、未開封の誕生日プレゼントを抱えていた。

もう胸に押しつけるようには持っていない。落とさないように、奪われないように、けれど、それだけではなく。自分がここから何を返すのか、まだ分からないまま、両手で確かめるように持っている。

中央の部屋には、棺の形をした炉があった。

その周りに、四つの扉がある。

前の部屋。後の部屋。左の部屋。右の部屋。どの扉にも鍵はかかっていない。真壁透一郎が言った通り、四方の部屋はもう開いている。開いている、というより、こちらが入るのを待っている。

炉の前に、真壁の姿はまだない。

スピーカーだけが鳴った。

『悲しみには、置き場所が必要です』

灯は答えない。

『あなたが持っているものも、あなたが抱えたままでは腐ってしまう。正しい火に返せば、形を変えられる』

「返すって、なに」

灯の声は小さい。けれど、さっきよりも少しだけ硬かった。

『失った人を、正しい位置へ戻すことです』

その言葉の奥で、別の部屋の祈りが強くなる。誰かの名前。誰かの命日。誰かが戻ればよかった日の話。大勢の人がいる。けれど、誰も武器を持って押し寄せてはこない。彼らは戦うためにここにいるのではない。祈るために、記録するために、見守るために、自分の喪失を炉へ差し出すために、ここにいる。

だからこそ、炉は動いている。

四方の部屋は、その願いを形にするための留め具だった。

複雑な鍵も、数字合わせもない。扉は最初から開いている。ここで必要なのは、手順を当てることではなく、部屋に置かれたものを見て、灯が何を思い出し、探索者が何を受け止めるかだった。

前の部屋には、小さな火種が置かれていた。

火種と言っても、炎は見えない。黒い皿の上に、赤くも白くもない点がひとつだけ残っている。目を逸らすと消え、見つめるとまぶたの裏でだけ明るくなる。そこへ近づくと、駅前の看板の一字、新聞の日付、スマホの地図がずれる感覚が、いっせいに戻ってきた。

探索者がそれを境界の熱だと気づくなら、灯は小さく頷く。

「あなたにも、ずれて見えてたんだ」

灯は、初めて少しだけ安心したような顔をする。自分だけがおかしいのではない。目の前の相手も、この世界のずれを見ている。だから火種は、ここではない場所への入口を示すものだと分かる。

後の部屋には、名札が並んでいた。

白い長机。空の椅子。椅子の前に、一枚ずつ名前の札が置かれている。どれも黒い文字で、丁寧に書かれている。けれど、中央の一枚だけが空白だった。名前がないのではない。まだ誰のものにもなっていない席として、空いている。

灯はその名札を見て、息を止めた。

「ここに座ったら、誰かの代わりになるのかな」

探索者が、代わりではないと伝えるなら、灯はすぐには返事をしない。ただ、空白の名札から少しだけ目を離す。呼ばれるのを待つ席ではなく、自分で立ち上がるために空けておく席として、その名札を見直す。

左の部屋には、止まった懐中時計があった。

蓋は開いている。針は同じ時刻を指したまま動かない。秒針だけが、進む直前の震えを繰り返している。耳を近づけても音はしない。けれど、その無音の中に、事故の日の白い息や、割れるガラスの気配が薄く混じっている。

灯は時計を見たまま、言う。

「わたし、あの日から進んでないのかもしれない」

探索者が急かさず隣にいるなら、灯は時計に触れる。進んでいないことを責めるためではなく、進ませるかどうかを自分で選ぶために。止まった懐中時計は、過去へ戻る道具ではなく、止まった時間を一度だけ見つめ直すためのものだと分かる。

右の部屋には、灰の小瓶があった。

透明な瓶の底に、薄い灰が溜まっている。ラベルには番号だけ。誰の遺品を焼いた灰なのかは書かれていない。書かれていないからこそ、ここに集められた喪失の多さが分かった。ひとつひとつの名前は、炉へ入れられると、番号と灰になる。

灯は瓶を持ち上げない。

「名前、消えちゃうんだ」

探索者が、消さないために見つけたのだと返すなら、灯はようやく瓶を見る。灰は終わったものではなく、誰かが勝手に終わらせたことの証拠だった。だから持ち出す。誰かの悲しみを盗むためではなく、番号にされた名前を、もう一度名前として扱うために。

探索者と灯は、四方の部屋で言葉を交わしながら、アーティファクトを持ち出す。

ひとつ目で、炉の底の音が変わる。

ふたつ目で、別室の祈りがわずかに乱れる。

誰かが名前を読み間違える。誰かが椅子を引く。紙が落ちる。磨りガラスの向こうで、人影が立ち上がり、また座る。けれど、彼らは襲ってこない。自分たちの祈りが何に使われているのか、半分も知らないまま、ただ不安そうに声を潜めている。

灯は唇を噛む。

「みんな、戻ってきてほしいんだ」

責める声ではない。

「でも、わたしは」

言葉はそこで途切れた。

三つ目を動かしたとき、炉が鳴った。

金属が軋むような音ではない。息を吸う音に似ていた。棺型の炉が、燃えるものを求めて、床の下から空気を吸い上げる。壁の円がゆっくりずれ、四方の扉の隙間から白い光が滲んだ。

スピーカーが、短く途切れる。

『やめなさい』

真壁の声ではなかった。

廊下の奥から、灯の親族夫婦が現れる。

空き家の玄関で聞いた声。帰りましょうと言った声。灯の身体が、言葉より先に拒んだ人たちの声。二人は走ってはこない。騒がない。まるで、迎えに来ただけだという顔をして、四方の部屋の前に立つ。

伯母は灯を見た。

「どうして、またこんなことをするの」

灯の手が、ライオンの袋を握る。

伯父は探索者を見た。

「その子を惑わせないでください。あなたは、その子の何でもない」

何でもない。

その言葉だけが、炉の音よりもはっきり響いた。

灯は一歩下がる。けれど、探索者の背に隠れない。隠れかけて、止まる。自分の足を見て、息を吸い直す。

「帰らない」

その声は震えていた。

「もう、あの家には帰らない」

伯母の顔から、困ったような笑みが消える。

「あの子を返してもらうまで、終われないの」

誰のことを言っているのか、説明は要らなかった。亡くした息子。戻らない子ども。灯に重ねられた、灯ではない誰か。帰還の環の炉へ差し出されたメダル。善意の形をしていた言葉の中に、ずっと隠れていた刃。

伯父が手を伸ばす。

灯を連れていくための手。

あるいは、炉へ戻すための手。

戦いは、長くはなかった。

親族夫婦は訓練された戦闘員ではない。けれど、灯を縛るための言葉と、ためらわせるための距離を知っている。名前を呼ぶ。心配を装う。あなたのためだと言う。あの子も望んでいると言う。灯の足が止まるたび、炉の火が近づく。

探索者が間に入る。

灯も、逃げるだけではいられない。

もみ合いの中で、祈りの声が途切れた。灰の小瓶が床を転がる。止まった懐中時計の蓋が鳴る。境界の火種が、まぶたの裏だけで大きく燃える。

最後に残ったのは、音だった。

倒れる音。

灯が息を飲む音。

探索者の手の中から、力が抜ける音。

親族夫婦は、もう立ち上がらない。

灯は、しばらく動かなかった。

彼女の目は、倒れた二人を見ている。けれど、何かを探している目ではない。奪うものを探している目でも、勝ったことを確かめる目でもない。ただ、自分が戻れない場所を本当に壊してしまったのだと、遅れて理解する目だった。

「わたしが」

声が出る。

「わたしが、やった」

探索者が何を言うかで、この場の温度は変わる。

灯一人のせいにすれば、彼女は頷くかもしれない。自分が悪いのだと、すぐに受け取ってしまうかもしれない。そういう形で重荷を持つことに、彼女は慣れすぎている。

けれど、そうではないと言うこともできる。

これは灯だけの行為ではない。探索者もここにいた。探索者も止めなかった。探索者も、生き延びるために選んだ。誰かを罰するためではなく、誰かの代わりになるためでもなく、二人でこの場所から出るために、取り返しのつかない線を越えた。

灯は、探索者の言葉を聞く。

何も許されない。

それでも、灯一人だけに背負わせることはできない。

「……うん」

灯は、かすかに頷いた。

「わたしだけじゃ、ない」

その言葉は救いではない。軽くもならない。けれど、灯がその場に崩れ落ちないための、細い手すりにはなった。

ここでは、結婚指輪には触れない。

親族夫婦の手元を見ることさえ、まだできない。何かを持っていこうという発想はない。儀式の材料を探す余裕もない。そこにあるのは、死と、息と、炉の音だけだった。

中央の炉で、真壁透一郎が待っている。

いつの間に現れたのか、分からなかった。

炉の向こう側に、真壁が立っている。白い手袋。整えられた髪。相談室で聞くなら安心してしまいそうな声。そのすべてが、地下のぬるい空気の中では、ひどく薄く見えた。

「悲しいことです」

真壁は、倒れた親族夫婦のほうを見なかった。

「けれど、儀式は続けられます。喪失は、喪失を呼ぶ。あなたがたが今作った空白も、炉は受け取ることができる」

灯の顔から血の気が引く。

「やめて」

「願いは個人のものではありません。ひとりが耐えられない悲しみを、輪が支える。そうして道が開く」

真壁の背後で、棺型の炉が開きかける。

中には、燃えきらないものがある。事故現場から盗まれたキーホルダー。未開封の誕生日プレゼントが置かれるべき場所。灰になったメダル。名前のない札。ここへ集められた、誰かの戻ってきてほしいという願い。

真壁はそれらを、悲しみを慰めるためだけに使っているのではない。

境界を観測するために使っている。

帰還という言葉で覆われた、別の実験。

探索者と灯が四方の留め具を乱したことで、炉は不安定になっている。真壁はその乱れを押さえ込もうとする。炉の円が回り、壁の影がずれ、同じ瞬間が二度重なって見える。灯が瞬きをするたび、空き家の玄関、事故の白い息、地下の炉が、薄い紙のように重なって剥がれた。

ここで、真壁と戦う。

勝ったなら、真壁は倒れるのではない。

儀式の制御を失い、炉の向こう側へ滑るように消える。白い手袋だけが一瞬、火のない火に照らされ、次の瞬間にはそこにいない。彼の声だけが、壁の奥に残る。

『道は、開きかけている』

負けたなら、真壁は勝ち誇らない。

境界の揺らぎが強くなり、探索者と灯の足元から時間がほどける。倒したはずの椅子が立ち、落ちたはずの灰の小瓶が床を転がり直し、聞いたはずの名前がもう一度呼ばれる。真壁はそのぶれの中へ身を隠す。追おうとしても、どこへ消えたのか分からない。

どちらにしても、真壁はいなくなる。

残るのは、無人の儀式場だった。

別室の祈りは遠くなっている。磨りガラスの向こうの人影も、もう見えない。彼らが逃げたのか、眠ったのか、境界のぶれに巻き込まれたのかは分からない。ただ、中央の炉の前に立っているのは探索者と灯だけだった。

灯は、ゆっくりと周りを見る。

四方の部屋。

乱された留め具。

棺型の炉。

盗まれた遺留品。

未開封のプレゼント。

「これ」

灯が言う。

「壊しただけじゃ、帰れないんだね」

炉はまだ息をしている。

儀式は止まりかけている。けれど、完全には消えていない。真壁が作ったものは、失った誰かをそのまま返すための道ではなかった。大勢の願いを燃やし、境界をこじ開けるための炉だった。

けれど、構造は残っている。

四方の部屋は、方角を作る。

遺留品は、帰る相手との結びつきを示す。

炉は、未練を送り火へ変える。

この儀式を、真壁の目的から切り離せるなら。

ひとりを呼び戻すのではなく、ここに迷い込んだ二人を、元の場所へ返す形へ組み替えられるなら。

探索者と灯は、四方のアーティファクトを炉の周りへ戻す手順を確かめる。すべてを戻してもいい。三つを戻し、ひとつだけを持ち出す余地を残してもいい。けれど、帰還の儀式を再現するには、少なくとも三つの方角が必要だった。

境界の火種は、道の入口を示す。

空席の名札は、呼び戻す相手ではなく、帰る席を空ける。

止まった懐中時計は、事故の日に縛られた時間を、もう一度だけ動かす。

灰の小瓶は、燃やされた遺品がまだ終わっていないことを示す。

けれど、中心に置くべきものは、まだ分からない。

二人分の帰還儀式にするためには、二人を一組として扱える媒介がいる。四方のアーティファクトだけでは場は作れる。ライオンのキーホルダーとプレゼントだけでは、灯と死んだ親友の未練に近すぎる。そこへ探索者が入ると、また「あの子」の代わりになってしまう。

灯は、探索者を見る。

「二人分なら」

自分で言って、そこで言葉を止めた。

まだ、答えは出ていない。

灯は、未開封の誕生日プレゼントを抱いたまま、炉を見る。

「あの子に渡すはずだったもの」

声は震えている。

けれど、そこで言葉は途切れる。今それをどう扱うべきかは、まだ分からない。答えを急げば、また誰かを誰かの代わりにしてしまう。

炉の底で、火のない火が揺れる。

真壁の儀式は壊れた。

けれど、帰るための送り火は、まだ作れる。

探索者と灯は、二人分の帰還儀式を始めるために、まだ足りない中心を探す。

## このシーンで扱うこと / 扱わないこと

- 四方の部屋は、教団儀式を成立させる留め具として扱う
- 四方の部屋のギミックは重い謎解きにしない。探索者と灯の会話で意味が見つかり、二人の距離が少しずつ縮まる場面として扱う
- 境界の火種、空席の名札、止まった懐中時計、灰の小瓶は、儀式妨害と帰還儀式再現の両方に関わる
- 帰還の環の多数構成員は、祈り手・記録係・見守り係として儀式を支えているが、直接戦闘員としては扱わない
- 親族夫婦は、儀式を乱されたことで灯を止めに来る直接の刃として現れる
- 親族夫婦との衝突後、二人は死亡する
- 親族夫婦の死亡直後には、結婚指輪を取得しない。指輪を儀式に使う発想も出さない
- 真壁戦後、勝敗にかかわらず真壁はいなくなる
- 無人の儀式場に探索者と灯だけが残ってから、初めて二人分の帰還儀式へ組み替える必要に気づく
- このシーンでは、結婚指輪をまだ取得しない。未開封の誕生日プレゼントを探索者の帰還燃料にする提案と探索者の拒否を経てから、次の送り火シーンで指輪に思い至る
- 灯一人に罪を背負わせず、探索者が共犯性を引き受けるかどうかを感情の分岐点にする

扱わない:

- 親族による加害の具体内容
- 親族夫婦の身体破壊の詳細描写
- 殺害を肯定する文脈
- 親族夫婦死亡直後の指輪取得
- 真壁戦前の指輪発想
- 未開封の誕生日プレゼントを帰還燃料にする案を断る前の指輪発想
- 多数の教団構成員を倒して突破する戦闘展開
- 真壁を完全に倒し切ったと断言する描写
- 結婚指輪だけで帰還が確定するという断言

## GM向け進行メモ

- 四方アーティファクトは二つ以上動かすと儀式妨害が成立する
- 各部屋はパズルの正解を探すより、灯の反応と探索者の返答でアーティファクトの意味が開く構成にする
- プレイヤーが迷い始めたら、灯の台詞で次に見るべき対象を示し、長時間の詰まりを作らない
- 帰還儀式再現には、三つ以上の四方アーティファクトと親族夫婦の結婚指輪が必要になる
- 四方アーティファクトを持ち帰らせる場合でも、帰還儀式に戻す数と持ち出せる数の制限を意識する
- 親族夫婦戦後は、すぐに指輪へ視線を向けない。まず「何をしてしまったか」「誰が背負うか」を置く
- 真壁戦は勝敗で結果の質感が変わるが、どちらでも `makabe_gone` へ進める
- 真壁が消えたあとの沈黙を長めに取り、ここでは「二人分に組み替える必要がある」までに留める
- 指輪取得はこのシーンで行わず、送り火シーンで未開封の誕生日プレゼントを探索者の帰還燃料にする案を断った後に思いつく
- 次の送り火シーンへつなぐため、未開封の誕生日プレゼントはまだ開けない
