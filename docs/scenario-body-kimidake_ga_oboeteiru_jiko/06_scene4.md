# 06 シーン4 迎えに来る家

最初に鳴ったのは、固定電話だった。

リビングの隅。低い棚の上に置かれた電話機。誰もいない家で、まだ回線だけが残されていることを、探索者も灯もそれまで意識していなかった。電子音は短く、規則正しく、冷えた部屋の空気を冷たく重く震わせる
灯の肩がくすむ。

膝に置いたライオンのキーホルダーを、手のひらで押さえる。落とさないためではない、縋るようににぎる
留守番電話に切り替わる。小さなノイズのあと、知らない女の声が入った。

『こちら、帰還の環、地域見守り窓口です。本日、駅前で水瀬灯さんと思われる方を見かけたというご連絡がありました。ご家族の方には、すでに共有しております。念のため、事故関係者宅の見回りも――』

そこで録音は途切れた。

部屋はまた静かになる。けれど、さっきまでの静けさではなかった。秒針の音が、急に細く尖って聞こえる。

灯は電話機を見ている。目を逸らせないでいる。

「わたし、見られてたんだ....」

声は小さかったが、事実を確かめるための声だった。

「あいつらが来るかもしれない」

帰還の環。

喪失者を支える会。献花台を整え、遺族の相談を受け、取り残された人に声をかける団体。そういう顔をしている。けれど、その顔の裏で、事故に関わった家、遺留品を持つ者、生還者の居場所、喪失を抱えた家族の名前が、丁寧に、善意の書式で記録されている。

駅前で見かけた誰かが、灯を心配して連絡したのかもしれない。

水瀬灯らしき少女がいた。汚れた服で、知らない人物と一緒だった。その人物は、亡くなった親友にどこか似ていた。

そういう言葉は、通報ではなく見守りとして渡される。心配だから。放っておけないから。助けたいから。その形をして、灯の逃げ場だけが削られていく。

窓の外で、車が一台止まった。

エンジンの音が切れる。ドアが開く音。閉まる音。もう一つ、少し遅れて閉まる音。灯はソファから立ち上がろうとして、立てなかった。膝の上の手が、コートの袖を探す。袖を握る手に力が入る。

インターホンが鳴る。

画面には、伯母が映っていた。薄い化粧。喪に近い色の服。背筋を伸ばし、玄関先で丁寧に立っている。画面の端に、伯父の肩が半分だけ入っている。

『灯ちゃん』

伯母の声は、スピーカー越しでも柔らかかった。

『迎えに来たわよ。寒かったでしょう。もう大丈夫だから、出ていらっしゃい』

灯は答えない。呼吸だけが浅くなる。吸う音が短い。吐く音が続かない。探索者のほうを見るが、助けて、と言う目ではなかった。言っていいのか分からない目だった。

伯母は画面の向こうで、少しだけ首を傾ける。

『帰還の環の方が教えてくださったの。駅前で見かけた方が心配して連絡してくださったそうよ。それに、ここはあの子のおうちでしょう。灯ちゃんが行くなら、ここだと思ったの』

あの子のおうち。

その言い方で、リビングの空気が一段冷える。ここは死んだ親友の家で、探索者の家ではない。灯がさっき、必死に自分に言い聞かせた言葉に、伯母は何のためらいもなく踏んでくる。

そこに伯父の声が割り込む。

『おい。誰かいるのか』

短い声。画面の中で、伯父が一歩前に出る。眉間に皺。首が太い。こちらを見ようとして、カメラに顔を近づけすぎる。

『開けろ。灯。聞こえてるだろ』

灯はソファの端に座ったまま、首を横に振った。最初は小さく。次に、もう少しはっきり。

「いや」

声は部屋の中でかすれた。

インターホン越しには届かないほどの声だった。それでも、灯は自分の口で言った。

伯母は微笑んだまま、チャイムをもう一度押す。電子音が重なる。固定電話がまた鳴る。玄関の外で、伯父が郵便受けを覗く音がする。金属の蓋が鳴る。善意の見守り、家族への共有、事故関係者宅の確認。そのすべてが、今は一人の少女を追い詰めている。

探索者が一歩動くと、灯が袖をつかんだ。

止めるためではない。

置いていかれないためでもない。

自分が立ち上がるために、そこにいる誰かの存在を確かめるような手だった。

「わたし」

灯は言う。声が震える。けれど、言葉は切れない。

「帰らない」

画面の向こうで、伯母の表情が少しだけ固まる。笑みの形は残ったまま、目の奥だけが動かなくなる。

『灯ちゃん。わがままを言わないで』

「帰らない」

今度は、少し大きかった。

伯父が玄関扉を叩く。強くはない。まだ、外から見れば訪問者の範囲に収まる強さ。けれど、その音で灯の指が探索者の袖に沈む。

探索者は灯の前に立つのではない。灯を隠すのでもない。立ち上がろうとする彼女の横に、同じ方向を向いて立つ。選ぶのは灯だ。ここから出るのも、ここに留まるのも、言葉を返すのも。

灯はもう一度、インターホンの画面を見る。

「わたしは、あなたたちの家に帰らない」

伯母の微笑みが消えた。

家の中で、秒針だけが進んでいる。帰ってこなかった人たちの家。そこへ、迎えに来たと言う人たちが立っている。迎えに来たのではない。連れ戻しに来たのだ。

窓の外で、伯父が勝手口のほうへ回る足音がする。

休息は終わった。

けれど、灯はまだ座っていない。震える膝で、立っている。
