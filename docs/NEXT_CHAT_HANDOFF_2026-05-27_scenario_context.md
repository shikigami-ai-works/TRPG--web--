NPCの未練:

NPCの基本設定:

- 名前は水瀬 灯（みなせ あかり）。
- NPCは主人公探索者と同い年の18歳。
- 平行世界側の探索者とは、同年代の親友として誕生日旅行に同行していた。
- 一人称は「わたし」。
- 主人公探索者への呼び方は基本的に「きみ」。
- 口調は短く砕けた同年代口調で、動揺時は言葉が途切れやすい。

NPCが主人公探索者に頼る理由:

- NPCは主人公探索者が死んだ親友本人ではないと頭では分かりかけている。
- それでも、魂の芯のようなものが親友と同じだと直感してしまう。
- 事故後の孤立と二次被害もあり、主人公探索者を唯一の避難先のように見てしまう。
- 救済には、主人公探索者を「死んだ親友の代わり」ではなく「別世界から来た別人」と受け入れる過程が必要。

灯の信頼度設計:

## 直接の敵と共犯関係

- シナリオ上の直接の敵は、NPCを引き取った親族。
- 親族は、主人公探索者を殺せば次こそ自分の息子が代わりに呼ばれると思い込んでいる。
- 親族の家庭で、主人公探索者を殺そうとした親族の母親をNPCが殺める。
- それを見て激昂した親族の父親を、主人公探索者が殺める。
- この事件により、主人公探索者とNPCは共犯になる。
- 二人は、自分たちが殺めてしまった親族夫婦の結婚指輪を使って、帰還のための儀式を再現する。

## 教団

- 教団名は「帰還の環（きかんのわ）」。
- 教団は、悲劇を経験し、失った人を取り戻したいと願う人々の集団。
- 死者そのものを蘇らせるのではなく、平行世界や境界の向こうから、ほぼ本人と呼べる存在を呼び戻そうとしている。
- 表向きは喪失や事故の当事者が集まる自助グループ。
- 裏では、世界の境界を越える儀式を行っている。
- 教団導師の目的は、時空をつかさどる神ヨグ・ソトースの先触れを研究し、交信し、召喚すること。
- 導師は信徒たちの「失った人を取り戻したい」という願いを利用している。
- 最終目的は、信徒全員を使った大儀式でヨグ・ソトースの先触れを召喚すること。

導師本人の仮設定:

- 名前は真壁透一郎（まかべ とういちろう）。
- 表向きの肩書きは、喪失者支援団体「帰還の環」代表。
- 元・大学非常勤講師で、専門は民俗学、宗教儀礼、死生観研究。
- アーキタイプは、善人の顔をした研究者型の搾取者。
- 外見は40代後半から50代前半、痩せ型、黒いスーツ、喪章めいた黒いリボン、手袋。
- 口調は静かで穏やか。共感を装い、悲しみや罪悪感を誘導する。
- NPCを「かわいそうな子」と呼ぶが、実際には境界接触者として見ている。
- 主人公探索者を、異世界から呼ばれた実例、または扉を通過した標本として扱う。
- 肉体的には弱いが、儀式場では炉、椅子、四方アーティファクト、信徒の祈りを使って空間を歪める。
- 実装IDは仮に `makabe_touichiro` とする。

## 遺留品とプレゼント

教団が持ち去った遺留品:

- おそろいのライオンキーホルダーの片割れ。
- 未開封の誕生日プレゼントの包み。

キーホルダー:

- モチーフはライオン。
- 主人公探索者は、かっこいいライオンが好き。
- NPCは、かわいいライオンが好き。
- NPCは片方を現在も持っている。
- もう片方は事故現場から教団に持ち去られた。

誕生日プレゼント:

- 中身は、ライオンのネックレスと短い恋文。
- ライオンのネックレスには、「ずっと一緒にいたい」「あなたを独占したい」「守りたい」という深い愛情と絆の意味を込める。
- 短い恋文は、NPCが親友へ渡せなかった告白として扱う。
- 開けるか、開けずに返すかは終盤の選択肢にできる。

## 儀式部屋

- 儀式の中心は、教団拠点内の、椅子がぽつんと一つだけ置かれた一室。
- 部屋には炉がある。
- 椅子は、呼び戻される者を迎える席であり、召喚儀式の焦点。
- 儀式では炉に火を入れ、遺品を投げ入れる。
- 実際の儀式構造は中心部屋だけで完結せず、炉の部屋の前後左右に配置された部屋の物品が関係している。

四方の部屋のアーティファクト:

- 共通方針として、四方アーティファクトは「便利だが、境界に触れた分だけ次周へ傷や汚染も残る」報酬にする。
- 役割は、境界察知、NPC攻略、判定救済、真相追跡に分ける。
- 前の部屋: 境界の火種。
  - 炉に火を入れるための火種。
  - 持ち帰ると、次回以降に境界現象を一度だけ事前察知できる。
  - 儀式、境界、異界化、平行世界転移の兆候を調べる判定に一度だけ+20の補正を得る。
  - 代償として、主人公探索者の境界侵食が+1される。
- 後の部屋: 空席の名札。
  - 呼び戻される者のための名前札。
  - 持ち帰ると、NPCの記憶継承を強化する。
  - 次周以降のNPC初期信頼度が+10される。
  - NPCの記憶断片と、特定選択肢「今度は一緒に帰ろう」が解禁される。
  - 代償として、NPCの悪夢レベルが+1される。
- 左の部屋: 止まった懐中時計。
  - 時空系のアーティファクト。
  - 判定のやり直し、または過去視に一度だけ使える。
  - 判定に使う場合、失敗した判定を1回だけ振り直せる。
  - 過去視に使う場合、事故、教団、親族、儀式に関する過去の断片を覗ける。
  - 代償として、使用後にSAN/精神値を1d3失う、または境界侵食が+1される。
- 右の部屋: 灰の小瓶。
  - 炉で焼かれた遺品の灰。
  - 持ち帰ると、真相解明や教団追跡のキーになる。
  - 教団、儀式、過去事件に関する調査判定に一度だけ+15の補正を得る。
  - 教団の別拠点、導師の過去、儀式に使われた遺留品の履歴など、次シナリオへの導線を解禁する。
  - 代償として、持ち主は死者の記憶を断片的に見てしまう。

## 勝利条件と帰還条件

四方の部屋の勝利条件:

- 四方の部屋からアーティファクトを2つ以上持ち去ることで、一時的に教団の儀式を邪魔できる。
- 導師を倒した後、アーティファクトを3つ以上もとの場所に戻すことで、主人公探索者とNPCが帰還のための儀式を再現できる。
- 持ち去りは妨害、返却は再現として扱う。
- 帰還時に最終的に持ち帰れる四方アーティファクトは1つまで。
- 2つ以上を持ち帰ろうとすると、帰還儀式を再現できなくなる。

導師を倒す条件:

- 儀式を妨害する。
- NPCの親戚を殺す。
- 戦闘で導師を倒す。
- 以上3つを満たすことで、導師撃破として扱う。

NPCを連れて帰る条件:

- 信頼度が一定値以上である。
- 恋文の入った誕生日プレゼントは開けない。
- NPCの未練を断ち、炉に儀式に使われたアイテムを入れる。
- 主人公探索者とNPCが、共犯の罪を共有する。
- 以上4つを満たすことで、NPCを元の世界へ連れて帰れる。

エンディング構成:

```yaml
endings:
  - id: return_with_akari
    name: 双つ灯の生還
    result: 二人で帰る

  - id: return_without_akari
    name: 空席に残る灯
    result: 一人で帰る

  - id: stay_with_akari
    name: 境界に灯るふたり
    result: 二人で残る
    memory_inheritance: true

  - id: boundary_collapse
    name: 境界の狭間に揺れる
    result: 帰還失敗
```

「境界に灯るふたり」は、主人公探索者が帰らず、あかりと共にこちらの世界に残留するエンド。帰還はしないが、あかりとの関係性は次周以降へメモリーとして引き継がれる。次周以降、あかりは初対面なのに置いていかれなかった感覚を持ち、「置いていかないで」系の会話や選択肢に変化が出る。

NPCを連れて帰れない場合:

- エンド名は「空席に残る灯」。
- 主人公探索者の帰還条件は満たしているが、NPC連れ帰り条件4つのうち1つ以上が欠けた場合に発生する。
- NPCは死亡せず、平行世界側に残る。
- 主人公探索者は元の世界へ戻るが、NPCは炉の前の椅子に座ったまま、境界を越えられない。
- NPCが主人公探索者を「死んだ親友の代わり」ではなく「別世界から来た別人」として受け入れきれない場合、境界を越える条件を満たせない。
- 特に、恋文の入った誕生日プレゼントを開けた場合は、NPCの中で主人公探索者が「親友の代用品」として固定される強い分岐として扱う。
- NPCは最後に「大丈夫。今度は、ちゃんと分かる気がする」と言い、次周への記憶継承を示唆する。
- 次周以降は、NPCの初期信頼度上昇、儀式部屋や炉に関する断片ヒント、特定選択肢「今度は一緒に帰ろう」が解禁される。

## 周回とアイテム枠

- プレイヤーには持ち帰りアイテム枠が2つある。
- 連絡先を受け取ることが可能なNPCにも、持ち帰りアイテム枠が2つある。
- 二週目以降、プレイヤーはアイテムを最大4つまで持ち込める。
- 持ち込みアイテム枠とは別に、装甲と武器を持てる。
- ここでいう装甲とは、「装甲値」が割り振られたアイテムを指す。
- 持ち帰り枠と持ち込み枠は別概念として扱う。

    armor: 1
