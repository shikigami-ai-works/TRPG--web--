# オリジナル短編シナリオ 決定要素メモ

## 位置づけ

このファイルは、MVP候補のオリジナル短編シナリオについて、会話で決まった物語要素をまとめたもの。
正式な `scenario.yaml` に落とす前の、設定・感情線・儀式構造の整理として扱う。

## シナリオの方向性

- 現代日本を舞台にした探索ホラー。
- 平行世界、夜行バス事故、NPCとの信頼攻略、教団儀式を中心にする。
- AI NPCとの関係性を資産化するゲーム方針に合わせ、NPCの未練と救済を物語の核に置く。
- 教団は完全な悪ではなく、喪失から救済を求めた人々が、他者の死の痕跡を利用してしまった存在として扱う。

## 仮題候補

- きみだけが覚えている事故
- 死んだきみの友達
- 空き家に帰る子
- あなたの世界で、きみは死んだ

## 導入の核

- 探索者は何らかのきっかけで平行世界へ迷い込む。
- その世界では、探索者の平行世界側の自分はすでに死亡している。
- 探索者は、死んだ自分の親友だったNPCと出会う。
- NPCの名前は水瀬 灯（みなせ あかり）。
- NPCは主人公探索者と同い年の18歳。
- 灯の一人称は「わたし」。
- 灯は主人公探索者を基本的に「きみ」と呼ぶ。
- 灯の口調は短く砕けた同年代口調で、動揺時は言葉が途切れやすい。
- NPCは探索者を見て、死んだ親友が戻ってきたと思い込み、強く動揺する。
- 探索者からするとNPCは知らない相手であり、強い戸惑いから物語が始まる。
- NPCは、主人公探索者が死んだ親友本人ではないと頭では分かりかけても、魂の芯のようなものが親友と同じだと直感してしまう。
- その直感により、NPCは主人公探索者を頼りたくなってしまう。

灯の口調サンプル:

```text
……なんで、そこにいるの。
```

```text
違うって分かってる。
分かってるのに、顔を見ると息ができなくなる。
```

```text
きみは、あの子じゃない。
……でも、置いていかないで。
```

```text
開けないで。
それは、あの子に渡すはずだったものだから。
```

```text
今度は、一緒に帰ろうって言って。
嘘でもいいから。
```

## 事故設定

- 平行世界側の探索者の家族と、NPCの家族は家族ぐるみの付き合いがあった。
- 平行世界側の探索者が18歳になる誕生日祝いとして、二家族で旅行が企画された。
- NPCも主人公探索者と同い年の18歳であり、同年代の親友として誕生日旅行に同行していた。
- 旅行の途中、夜行バスに乗ったところで事故に遭った。
- 事故により、平行世界側の探索者の家族、NPCの両親、平行世界側の探索者が死亡した。
- NPCだけが生き残った。
- 事故から現在までは、ほぼ49日が経過している。
- そのため、現在のNPCは「今日はきみの誕生日だった」とは言わない。
- 誕生日は現在の日付ではなく、「あの日、祝えなかった誕生日」として扱う。

## 事故後の経過

- NPCは事故後、10日ほど入院した。
- 退院後、平行世界側の探索者の家族とNPCの両親、両家の葬式が終わった。
- NPCは親族に引き取られた。
- 引き取り先の親族は、事故で息子を失っている。
- その親族は教団とつながっている人物であり、同時にNPCを傷つける加害者でもある。
- 親族は、事故で失った息子が帰ってくることを願って教団儀式に加担していた。
- 親族は、息子の遺品である中学生時代の大会メダルを儀式に使っていた。
- メダルは地区大会以上の、かなり大きな大会で得た賞として扱う。
- 親族は、儀式によって息子か、事故で死亡した平行世界側の探索者のどちらか片方が呼ばれるだろうと期待していた。
- しかし実際に呼ばれたのは親族の息子ではなく、別世界から来た主人公探索者だった。
- 事故からひと月ほど経ったころ、NPCは引き取り先で性的暴行を受けそうになった。
- 親族による性加害は、息子を失った悲嘆や教団儀式の目的とは別の加害として扱う。
- 親族はNPCに対して、ただの性加害者である。
- 親族の喪失や教団への加担は、性加害の理由づけや免罪にしない。
- NPCは事故の傷も癒えないうちにさらに傷つけられ、心身ともに限界に近い状態になっている。
- 現在のNPCは、主人公探索者以外に頼れる相手がほとんどいない。
- この出来事は、NPCの孤立、切迫、探索者への依存を支える背景として扱う。
- シナリオ中では露悪的な描写を避け、直接的な描写よりも、NPCの回避反応、生活環境、言葉の詰まり、助けを求められない状態として表現する。

## NPCの未練

NPCの未練は次で確定する。

> 親友だけが自分を助けて死んだと思っていた。

- NPCは、親友に誕生日プレゼントを渡せなかった。
- NPCは、親友の死を受け止めきれていない。
- NPCは、探索者を死んだ親友そのものとして見たい気持ちと、別人だと理解しなければならない現実の間で揺れる。
- NPCは、主人公探索者の中に親友と同じ魂の芯のようなものを感じ取り、理屈ではなく直感で頼ってしまう。
- NPCは、事故後の孤立と二次被害により、主人公探索者を唯一の避難先のように見てしまう。
- NPCの救済には、探索者を「死んだ親友の代わり」ではなく「別世界から来た別人」と受け入れる過程が必要になる。
- ただし、魂の芯が同じだと感じることは、主人公探索者が親友本人であることの証明ではない。

## 灯の信頼度設計

基本値:

- 信頼度の最大値は100。
- 初期信頼度は25。
- NPC連れ帰りに必要な信頼度は70。
- 空席の名札を持ち帰っている場合、次周以降の初期信頼度に+10する。

信頼度が上がる行動:

- `say_not_replacement`: +10
  - 主人公探索者が「自分は死んだ親友の代わりではない」と正直に伝える。
  - 灯を突き放すのではなく、別人として向き合う行動。
- `respect_gift_unopened`: +15
  - 誕生日プレゼントを勝手に開けず、灯が渡せなかった気持ちを尊重する。
  - 中身を知ることより、灯の未練を勝手に暴かないことを重視する。
- `protect_akari_without_possessing`: +10
  - 灯を守るが、所有物や保護対象として支配しない。
  - 灯自身の意思を残したまま危機から助ける。
- `let_akari_speak_regret`: +10
  - 灯が未練や後悔を話す時間を与える。
  - 無理に励ますのではなく、灯が自分の言葉で整理するのを待つ。
- `share_guilt_truthfully`: +15
  - 共犯になった罪を隠さず、灯と共有する。
  - 灯だけに罪を背負わせず、二人で生き延びるための事実として引き受ける。
- `promise_return_together`: +10
  - 「今度は一緒に帰ろう」と約束する。
  - 灯を残して帰るエンドや記憶継承後に特に効く選択肢として扱う。

信頼度が下がる行動:

- `act_as_dead_friend`: -15
  - 悪意を持って偽るのではなく、灯の心に寄り添いすぎて、つい死んだ親友本人として振る舞ってしまう。
  - 灯を安心させたい気持ちから「親友の代わり」の位置に入ってしまうが、結果的に灯が別人として受け入れる過程を妨げる。
- `open_birthday_gift`: -10
  - 誕生日プレゼントを開けてしまう。
  - 信頼度下降は-10にする。ただし、NPC連れ帰り条件では「恋文の入った誕生日プレゼントは開けない」を別フラグとして扱う。
- `dismiss_akari_regret`: -10
  - 灯の未練や後悔を軽く扱う。
  - 「もう終わったこと」「気にしなくていい」など、灯が抱えているものを否定する行動。
- `abandon_akari_in_crisis`: -20
  - 灯が危ない時に見捨てる。
  - 親族宅、教団拠点、儀式部屋など、灯が明確に助けを必要としている場面で発生する。
- `make_akari_bear_guilt`: -20
  - 共犯の罪を共有せず、灯だけに背負わせる。
  - 主人公探索者が自分の手を汚すことや責任を引き受けることを避け、結果として灯だけが罪悪感を抱える状態にする。

実装用メモ:

```yaml
npc_trust:
  max: 100
  initial: 25
  return_required: 70
  memory_bonus_empty_nameplate: 10

trust_gain:
  say_not_replacement: 10
  respect_gift_unopened: 15
  protect_akari_without_possessing: 10
  let_akari_speak_regret: 10
  share_guilt_truthfully: 15
  promise_return_together: 10

trust_loss:
  act_as_dead_friend: -15
  open_birthday_gift: -10
  dismiss_akari_regret: -10
  abandon_akari_in_crisis: -20
  make_akari_bear_guilt: -20
```

## 教団設定

- 教団名は「帰還の環（きかんのわ）」にする。
- 教団は、悲劇を経験し、失った人を取り戻したいと願う人々の集団。
- 教団の中心思想は、「失った人を取り戻したい」という願いに置く。
- 教団は死者そのものを蘇らせるのではなく、平行世界や境界の向こうから、ほぼ本人と呼べる存在を呼び戻そうとしている。
- その思想は喪失への祈りから始まっているが、他者の遺留品や未練を勝手に使う時点で歪んでいる。
- 表向きは、喪失や事故の当事者が集まる自助グループ。
- 裏では、世界の境界を越える儀式を行っている。
- 彼らの儀式が不完全に成立した結果、夜行バス事故と平行世界転移が発生した。
- 教団は事故後、現場に残された遺留品が境界の影響を帯びたことに気づいた。
- 教団の回収係は、警察や遺族より前に事故現場から一部の遺留品をこっそり持ち去った。
- 教団はそれらの遺留品を、再儀式のための燃料または媒介として利用した。
- NPCを引き取った親族は、事故で息子を失った遺族であり、教団とつながっている。
- この親族は、NPCの保護者という表向きの立場を利用して、教団側へ情報や身柄を流す可能性がある。
- 親族は息子を取り戻すため、息子の遺品である中学生時代の大会メダルを儀式に差し出していた。
- 親族にとって主人公探索者の出現は、儀式が成功した証であると同時に、息子が戻らなかった失敗の証でもある。
- ただし、親族の性加害は教団儀式の一部ではなく、親族個人の加害性として切り分ける。

### 教団導師の目的

- 教団導師の目的は、時空をつかさどる神ヨグ・ソトースの先触れを研究し、交信し、召喚すること。
- 導師は、信徒たちの「失った人を取り戻したい」という願いを利用している。
- 信徒たちには、儀式が失った人を呼び戻すためのものだと説明している。
- しかし導師の最終目的は、教団に集まった信徒たち全員を使った大儀式で、ヨグ・ソトースの先触れを召喚すること。
- 「帰還の環」は、表向きには喪失した人との再会を目指す教団だが、裏では先触れ召喚のための実験場になっている。

### 教団導師の仮設定

導師本人のキャラは仮決めとして、いったん次の設定で先に進める。

- 名前は真壁透一郎（まかべ とういちろう）。
- 表向きの肩書きは、喪失者支援団体「帰還の環」代表。
- 元・大学非常勤講師で、専門は民俗学、宗教儀礼、死生観研究。
- アーキタイプは、善人の顔をした研究者型の搾取者。
- 信徒には「失った人を取り戻す」と語るが、実際には悲しみを燃料として境界観測実験を進めている。
- 外見は40代後半から50代前半、痩せ型、背筋が妙にまっすぐな人物。
- 黒いスーツに喪章めいた黒いリボンをつけ、素手で遺品に触れないよう手袋をしている。
- 儀式時は白い布を肩にかけ、喪服と祭服の中間のような姿になる。
- 口調は静かで穏やか。共感を装いながら、相手の悲しみや罪悪感を誘導する。
- NPCのことは「かわいそうな子」として扱うが、本質的には事故を生き残った貴重な境界接触者として見ている。
- 主人公探索者のことは、異世界から呼ばれた実例、あるいは扉を通過した標本として扱う。
- 肉体的には強くないが、儀式場では炉、椅子、四方アーティファクト、信徒の祈りを使って空間を歪める。
- 戦闘・儀式演出には、時間の巻き戻り、死者の声、空席の影、境界の門の気配を使う。

想定台詞:

## 直接の敵と共犯関係

- シナリオ上の直接の敵は、NPCを引き取った親族にする。
- 親族は、主人公探索者を殺せば、次こそ自分の息子が代わりに呼ばれると思い込んでいる。
- そのため親族は、主人公探索者を殺すつもりで襲いかかってくる。
- 親族の家庭で、主人公探索者を殺そうとした親族の母親を、NPCが殺めてしまう。
- それを見て激昂した親族の父親を、主人公探索者が殺めてしまう。
- この事件により、主人公探索者とNPCは共犯になる。
- これは、主人公探索者とNPCが「手を汚してでも生き延びなければならない探索者」になる始まりの事件として扱う。
- 主人公探索者とNPCは、自分たちが殺めてしまった親族二人の遺品を使って、儀式を再現する。
- 儀式再現に使う親族二人の遺品は、親族夫婦の結婚指輪にする。
- 結婚指輪はペアの遺品であり、二人をひとつの関係として儀式へ結びつける媒介になる。
- 儀式再現は、罪の正当化ではなく、生き延びるために境界現象を逆利用する行為として扱う。

## 教団が持ち去った遺留品

教団が使う物品は、最初から魔法道具だったわけではない。
事故、未練、境界現象、教団儀式の影響によって歪められた「事故現場の遺留品」として扱う。

### おそろいのキーホルダー

- 平行世界側の探索者とNPCが持っていた、おそろいのキーホルダー。
- キーホルダーのモチーフはライオン。
- 主人公探索者は、かっこいいライオンが好き。
- NPCは、かわいいライオンが好き。
- 同じライオンモチーフでも、二人の好みの違いが関係性の温度差や親しさを示す。
- NPCは片方を現在も持っている。
- もう片方は事故現場から教団に持ち去られた。
- 2つは、親友同士のつながりを示す重要な証拠になる。
- 2つが近づくと、震える、冷たくなる、音が鳴る、光るなどの異常反応を起こす可能性がある。
- 終盤で、NPCが自分のキーホルダーを握ったまま、盗まれた片割れを儀式の炎へ返す展開が有力。

### 誕生日プレゼントの包み

- 誕生日旅行で渡されるはずだったプレゼント。
- 中身は、ライオンのネックレスと短い恋文。
- 主人公探索者はかっこいいライオンが好きだが、NPCはかわいいライオンが好きなため、ネックレスはなんとなくかわいい雰囲気のデザインになっている。
- ライオンのネックレスには、「ずっと一緒にいたい」「あなたを独占したい」「守りたい」という深い愛情と絆の意味を込める。
- 短い恋文は、NPCが親友へ渡せなかった告白として扱う。
- 未開封のまま事故現場に残された。
- 誰から誰へのプレゼントかは、序盤では明かしすぎない。
- 教団はこれを「渡されなかった祝福」「叶わなかった未来の器」として儀式に使った。
- 開けるか、開けずに返すかは終盤の選択肢にできる。
- どちらの場合でも、目的は思い出の否定ではなく、教団が勝手に縛った未練をほどくことにする。

## 儀式と帰還条件

元の世界へ戻るには、教団が儀式に使った遺留品を取り戻し、未練を断ち切るように儀式の炎で燃やして消し去る。

ただし、燃やす意味は「思い出を消す」ことではない。

> 教団が勝手に縛り直した未練を、送り火として焼きほどく。

儀式の炎を起こす場所:

- 教団拠点内の、椅子がぽつんと一つだけ置かれた一室。
- 部屋には炉がある。
- 椅子は、呼び戻される者を迎える席であり、同時に召喚儀式の焦点として扱う。
- 儀式では炉に火を入れ、遺品を投げ入れる。
- 実際の儀式構造はこの部屋一つだけで完結せず、炉の部屋の前後左右に配置された部屋の物品が関係している。
- 中心の炉部屋は儀式の焦点であり、前後左右の部屋は境界を支える補助点として扱う。
- 前後左右の部屋に置かれた物品を調べる、動かす、回収する、壊す、または正しい位置へ戻すことで、儀式の成立条件や帰還条件に影響を与える。
- 前後左右の部屋には、持ち帰れる可能性のあるアーティファクトを配置する。
- アーティファクトは、帰還後にNPCとの記憶、関係値、次シナリオへの引き継ぎ、または特殊選択肢に影響する報酬候補として扱う。
- ただし、すべてを安全に持ち帰れるとは限らず、持ち帰りには信頼度、儀式の安定度、選択肢、または代償を絡める。
- 部屋は祭壇然としすぎず、むしろ日常から切り離された空白のような印象にする。

四方の部屋のアーティファクト:

共通方針:

- 四方アーティファクトは、便利だが、境界に触れた分だけ次周へ傷や汚染も残る設計にする。
- 4つはそれぞれ、境界察知、NPC攻略、判定救済、真相追跡の役割を持つ。
- 帰還時に最終的に持ち帰れる四方アーティファクトは1つまでにするため、どれを次周へ残すかが攻略上の選択になる。

- 前の部屋: 境界の火種。
  - 炉に火を入れるための火種。
  - 持ち帰ると、次回以降に境界現象を一度だけ事前察知できる。
  - 儀式、境界、異界化、平行世界転移の兆候を調べる判定に一度だけ+20の補正を得る。
  - 代償として、主人公探索者の境界侵食が+1される。
  - 主人公探索者が境界の向こう側に少しだけ覚えられるリスクを持つ。
- 後の部屋: 空席の名札。
  - 呼び戻される者のための名前札。
  - 持ち帰ると、NPCの記憶継承を強化する。
  - 次周以降のNPC初期信頼度が+10される。
  - NPCの記憶断片と、特定選択肢「今度は一緒に帰ろう」が解禁される。
  - 代償として、NPCの悪夢レベルが+1される。
  - NPCは事故、炉、椅子、空席に関する夢を見るようになる。
- 左の部屋: 止まった懐中時計。
  - 時空系のアーティファクト。
  - 判定のやり直し、または過去視に一度だけ使える。
  - 判定に使う場合、失敗した判定を1回だけ振り直せる。
  - 過去視に使う場合、事故、教団、親族、儀式に関する過去の断片を覗ける。
  - 代償として、使用後にSAN/精神値を1d3失う、または境界侵食が+1される。
  - 時空をいじる報酬として強力だが、四方アーティファクトの中でも特に危険なものとして扱う。
- 右の部屋: 灰の小瓶。
  - 炉で焼かれた遺品の灰。
  - 持ち帰ると、真相解明や教団追跡のキーになる。
  - 教団、儀式、過去事件に関する調査判定に一度だけ+15の補正を得る。
  - 教団の別拠点、導師の過去、儀式に使われた遺留品の履歴など、次シナリオへの導線を解禁する。
  - 代償として、持ち主は死者の記憶を断片的に見てしまう。
  - 情報価値は高いが、他者の死の痕跡を抱えて帰る報酬として扱う。

四方の部屋の勝利条件:

- 四方の部屋からアーティファクトを2つ以上持ち去ることで、一時的に教団の儀式を邪魔できる。
- 儀式を邪魔した状態で、導師を倒す。
- 導師を倒した後、アーティファクトを3つ以上もとの場所に戻すことで、主人公探索者とNPCが帰還のための儀式を再現できる。
- アーティファクトを持ち去る行為は妨害、もとの場所に戻す行為は再現として扱う。
- 2つ以上の持ち去りと3つ以上の返却は、クライマックスの進行フラグとして扱う。
- 帰還時に最終的に持ち帰れるアーティファクトは1つまで。
- 2つ以上を持ち帰ろうとすると、帰還儀式を再現できなくなる。
- そのため、クライマックス中は2つ以上を一時的に持ち去って妨害し、その後3つ以上を戻し、残った1つだけを報酬として持ち帰る構造にする。
- システム上、プレイヤーには持ち帰りアイテム枠が2つある。
- 連絡先を受け取ることが可能なNPCにも、持ち帰りアイテム枠が2つある。
- ただし、このシナリオの四方アーティファクトは3つ以上を戻さないと帰還儀式を再現できないため、四方アーティファクトから最終的に持ち帰れるのは1つまでになる。
- 残りの持ち帰り枠は、別種の報酬、通常アイテム、記憶品、または次シナリオ用の取得物に使える。

導師を倒す条件:

- 儀式を妨害する。
- NPCの親戚を殺す。
- 戦闘で導師を倒す。
- 以上3つを満たすことで、導師撃破として扱う。

帰還条件の方向性:

- 教団が持ち去った遺留品を取り戻す。
- NPCが親友の死と、自分の未練を言葉にする。
- 探索者が「自分は死んだ親友の代わりではない」と宣言する。
- おそろいのキーホルダーの片割れ、誕生日プレゼントの包みなどを儀式の炎へ返す。
- 主人公探索者とNPCが殺めてしまった親族二人の遺品、親族夫婦の結婚指輪を使い、教団儀式の構造を再現する。
- 遺留品を燃やすことで、教団が作った境界の錨を消す。
- 境界の錨が消えることで、探索者は元の世界へ戻れる。

NPCを連れて帰る条件:

- 信頼度が一定値以上である。
- 恋文の入った誕生日プレゼントは開けない。
- NPCの未練を断ち、炉に儀式に使われたアイテムを入れる。
- 主人公探索者とNPCが、共犯の罪を共有する。
- 以上4つを満たすことで、NPCを元の世界へ連れて帰れる。

エンディング構成:

- エンディングの形は4つにする。
- あかりを連れて二人で帰る。
- あかりを残して主人公探索者だけが帰る。
- あかりと共にこちらの世界に残留する。
- 境界崩壊により帰還に失敗する。

```yaml
endings:
  - id: return_with_akari
    name: 双つ灯の生還
    result: 二人で帰る

  - id: return_without_akari
    name: 空席に残る灯
    result: 一人で帰る

  - id: stay_with_akari
    name: 境界に灯るふたり
    result: 二人で残る
    memory_inheritance: true

  - id: boundary_collapse
    name: 境界の狭間に揺れる
    result: 帰還失敗
```

あかりと共に残るエンド:

- エンド名は「境界に灯るふたり」にする。
- 主人公探索者が元の世界へ帰らず、あかりと共にこちらの世界に残留するエンド。
- 条件は、主人公探索者が「帰らない」と選び、あかりが生存していること。
- 帰還はしないが、あかりとの関係性は次周以降へメモリーとして引き継がれる。
- 引き継がれるメモリーは、「一緒に残ることを選んだ記憶の残響」として扱う。
- 次周以降、あかりは主人公探索者に対して、初対面なのに置いていかれなかった感覚を持つ。
- 「置いていかないで」系の会話や選択肢に変化が出る。

```yaml
ending:
  id: stay_with_akari
  name: 境界に灯るふたり
  result:
    player_returns: false
    npc_returns: false
    player_stays_with_npc: true
    npc_memory_inheritance: true
    relationship_asset_preserved: true
  unlocks:
    - stayed_together_memory
    - abandonment_fear_softened
    - boundary_residue_trace
```

灯を残して帰るエンド:

- エンド名は「空席に残る灯」にする。
- 主人公探索者の帰還条件は満たしているが、NPC連れ帰り条件4つのうち1つ以上が欠けた場合に発生する。
- NPCは死亡せず、平行世界側に残る。
- 主人公探索者は元の世界へ戻るが、NPCは炉の前の椅子に座ったまま、境界を越えられない。
- NPCが主人公探索者を「死んだ親友の代わり」ではなく「別世界から来た別人」として受け入れきれない場合、境界を越える条件を満たせない。
- 特に、恋文の入った誕生日プレゼントを開けた場合は強い分岐として扱う。恋文を読んでしまうことで、NPCの中で主人公探索者が「別人」ではなく「親友の代用品」として固定される。
- NPCは最後に「大丈夫。今度は、ちゃんと分かる気がする」と言い、次周への記憶継承を示唆する。
- このエンドは完全な失敗ではなく、NPCとの関係性が次周へ資産として残る記憶継承エンドとして扱う。

「空席に残る灯」の発生条件:

- 信頼度が不足している。
- 恋文の入った誕生日プレゼントを開けてしまった。
- NPCの未練を断てていない。
- 主人公探索者とNPCが、共犯の罪を共有できていない。
- 以上のうち1つ以上を満たす。

「空席に残る灯」の記憶継承報酬:

- 次周以降のNPCは、初対面でも主人公探索者を少しだけ怖がらない。
- NPCが「前にもここで会った気がする」といった断片的な反応を見せる。
- 儀式部屋、炉、椅子、または空席の名札に関するヒントが解禁される。
- NPC信頼度の初期値が少し上がる。
- 特定選択肢「今度は一緒に帰ろう」が解禁される。

## 周回と持ち込み枠

- 二週目以降のために、プレイヤーはアイテムを最大4つまで持ち込める。
- 持ち込みアイテム枠とは別に、装甲と武器を持つことができる。
- ここでいう装甲とは、「装甲値」が割り振られたアイテムを指す。
- 持ち帰り枠と持ち込み枠は別概念として扱う。
- 持ち帰り枠はシナリオ終了時に何を持ち帰れるかを決める枠。
- 持ち込み枠は二週目以降の開始時に何を持ち込めるかを決める枠。

## 終盤の感情線

- NPCは、親友が帰ってきたのではないと認めなければならない。
- 探索者は、死んだ平行世界側の自分の代わりにはなれないと伝えなければならない。
- 教団は、救われたい願いのために他者の遺留品と未練を利用したことを突きつけられる。
- 主人公探索者とNPCは、互いに相手を守るために人を殺めてしまった事実を抱える。
- 二人は、被害者であると同時に加害の記憶を共有した共犯として、帰還か残留かを選ばなければならない。
- 遺留品を燃やす場面は破壊ではなく、返還と送り火として描く。

```text
忘れるんやない。

```
