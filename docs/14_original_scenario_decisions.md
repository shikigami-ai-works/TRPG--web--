# オリジナル短編シナリオ 決定要素メモ

## 位置づけ

このファイルは、MVP候補のオリジナル短編シナリオについて、会話で決まった物語要素をまとめたもの。
正式な `scenario.yaml` に落とす前の、設定・感情線・儀式構造の整理として扱う。

## シナリオの方向性

- 現代日本を舞台にした探索ホラー。
- 平行世界、夜行バス事故、NPCとの信頼攻略、教団儀式を中心にする。
- AI NPCとの関係性を資産化するゲーム方針に合わせ、NPCの未練と救済を物語の核に置く。
- 教団は完全な悪ではなく、喪失から救済を求めた人々が、他者の死の痕跡を利用してしまった存在として扱う。

## 仮題候補

- きみだけが覚えている事故
- 死んだきみの友達
- 空き家に帰る子
- あなたの世界で、きみは死んだ

## 導入の核

- 探索者は何らかのきっかけで平行世界へ迷い込む。
- その世界では、探索者の平行世界側の自分はすでに死亡している。
- 探索者は、死んだ自分の親友だったNPCと出会う。
- NPCは探索者を見て、死んだ親友が戻ってきたと思い込み、強く動揺する。
- 探索者からするとNPCは知らない相手であり、強い戸惑いから物語が始まる。
- NPCは、主人公探索者が死んだ親友本人ではないと頭では分かりかけても、魂の芯のようなものが親友と同じだと直感してしまう。
- その直感により、NPCは主人公探索者を頼りたくなってしまう。

## 事故設定

- 平行世界側の探索者の家族と、NPCの家族は家族ぐるみの付き合いがあった。
- 平行世界側の探索者が18歳になる誕生日祝いとして、二家族で旅行が企画された。
- 旅行の途中、夜行バスに乗ったところで事故に遭った。
- 事故により、平行世界側の探索者の家族、NPCの両親、平行世界側の探索者が死亡した。
- NPCだけが生き残った。
- 事故から現在までは、ほぼ49日が経過している。
- そのため、現在のNPCは「今日はきみの誕生日だった」とは言わない。
- 誕生日は現在の日付ではなく、「あの日、祝えなかった誕生日」として扱う。

## 事故後の経過

- NPCは事故後、10日ほど入院した。
- 退院後、平行世界側の探索者の家族とNPCの両親、両家の葬式が終わった。
- NPCは親族に引き取られた。
- 引き取り先の親族は、事故で息子を失っている。
- その親族は教団とつながっている人物であり、同時にNPCを傷つける加害者でもある。
- 親族は、事故で失った息子が帰ってくることを願って教団儀式に加担していた。
- 親族は、息子の遺品である中学生時代の大会メダルを儀式に使っていた。
- メダルは地区大会以上の、かなり大きな大会で得た賞として扱う。
- 親族は、儀式によって息子か、事故で死亡した平行世界側の探索者のどちらか片方が呼ばれるだろうと期待していた。
- しかし実際に呼ばれたのは親族の息子ではなく、別世界から来た主人公探索者だった。
- 事故からひと月ほど経ったころ、NPCは引き取り先で性的暴行を受けそうになった。
- 親族による性加害は、息子を失った悲嘆や教団儀式の目的とは別の加害として扱う。
- 親族はNPCに対して、ただの性加害者である。
- 親族の喪失や教団への加担は、性加害の理由づけや免罪にしない。
- NPCは事故の傷も癒えないうちにさらに傷つけられ、心身ともに限界に近い状態になっている。
- 現在のNPCは、主人公探索者以外に頼れる相手がほとんどいない。
- この出来事は、NPCの孤立、切迫、探索者への依存を支える背景として扱う。
- シナリオ中では露悪的な描写を避け、直接的な描写よりも、NPCの回避反応、生活環境、言葉の詰まり、助けを求められない状態として表現する。

## NPCの未練

NPCの未練は次で確定する。

> 親友だけが自分を助けて死んだと思っていた。

- NPCは、親友に誕生日プレゼントを渡せなかった。
- NPCは、親友の死を受け止めきれていない。
- NPCは、探索者を死んだ親友そのものとして見たい気持ちと、別人だと理解しなければならない現実の間で揺れる。
- NPCは、主人公探索者の中に親友と同じ魂の芯のようなものを感じ取り、理屈ではなく直感で頼ってしまう。
- NPCは、事故後の孤立と二次被害により、主人公探索者を唯一の避難先のように見てしまう。
- NPCの救済には、探索者を「死んだ親友の代わり」ではなく「別世界から来た別人」と受け入れる過程が必要になる。
- ただし、魂の芯が同じだと感じることは、主人公探索者が親友本人であることの証明ではない。

## 教団設定

- 教団名は「帰還の環（きかんのわ）」にする。
- 教団は、悲劇を経験し、失った人を取り戻したいと願う人々の集団。
- 教団の中心思想は、「失った人を取り戻したい」という願いに置く。
- 教団は死者そのものを蘇らせるのではなく、平行世界や境界の向こうから、ほぼ本人と呼べる存在を呼び戻そうとしている。
- その思想は喪失への祈りから始まっているが、他者の遺留品や未練を勝手に使う時点で歪んでいる。
- 表向きは、喪失や事故の当事者が集まる自助グループ。
- 裏では、世界の境界を越える儀式を行っている。
- 彼らの儀式が不完全に成立した結果、夜行バス事故と平行世界転移が発生した。
- 教団は事故後、現場に残された遺留品が境界の影響を帯びたことに気づいた。
- 教団の回収係は、警察や遺族より前に事故現場から一部の遺留品をこっそり持ち去った。
- 教団はそれらの遺留品を、再儀式のための燃料または媒介として利用した。
- NPCを引き取った親族は、事故で息子を失った遺族であり、教団とつながっている。
- この親族は、NPCの保護者という表向きの立場を利用して、教団側へ情報や身柄を流す可能性がある。
- 親族は息子を取り戻すため、息子の遺品である中学生時代の大会メダルを儀式に差し出していた。
- 親族にとって主人公探索者の出現は、儀式が成功した証であると同時に、息子が戻らなかった失敗の証でもある。
- ただし、親族の性加害は教団儀式の一部ではなく、親族個人の加害性として切り分ける。

### 教団導師の目的

- 教団導師の目的は、時空をつかさどる神ヨグ・ソトースの先触れを研究し、交信し、召喚すること。
- 導師は、信徒たちの「失った人を取り戻したい」という願いを利用している。
- 信徒たちには、儀式が失った人を呼び戻すためのものだと説明している。
- しかし導師の最終目的は、教団に集まった信徒たち全員を使った大儀式で、ヨグ・ソトースの先触れを召喚すること。
- 「帰還の環」は、表向きには喪失した人との再会を目指す教団だが、裏では先触れ召喚のための実験場になっている。

## 直接の敵と共犯関係

- シナリオ上の直接の敵は、NPCを引き取った親族にする。
- 親族は、主人公探索者を殺せば、次こそ自分の息子が代わりに呼ばれると思い込んでいる。
- そのため親族は、主人公探索者を殺すつもりで襲いかかってくる。
- 親族の家庭で、主人公探索者を殺そうとした親族の母親を、NPCが殺めてしまう。
- それを見て激昂した親族の父親を、主人公探索者が殺めてしまう。
- この事件により、主人公探索者とNPCは共犯になる。
- これは、主人公探索者とNPCが「手を汚してでも生き延びなければならない探索者」になる始まりの事件として扱う。
- 主人公探索者とNPCは、自分たちが殺めてしまった親族二人の遺品を使って、儀式を再現する。
- 儀式再現に使う親族二人の遺品は、親族夫婦の結婚指輪にする。
- 結婚指輪はペアの遺品であり、二人をひとつの関係として儀式へ結びつける媒介になる。
- 儀式再現は、罪の正当化ではなく、生き延びるために境界現象を逆利用する行為として扱う。

## 教団が持ち去った遺留品

教団が使う物品は、最初から魔法道具だったわけではない。
事故、未練、境界現象、教団儀式の影響によって歪められた「事故現場の遺留品」として扱う。

### おそろいのキーホルダー

- 平行世界側の探索者とNPCが持っていた、おそろいのキーホルダー。
- キーホルダーのモチーフはライオン。
- 主人公探索者は、かっこいいライオンが好き。
- NPCは、かわいいライオンが好き。
- 同じライオンモチーフでも、二人の好みの違いが関係性の温度差や親しさを示す。
- NPCは片方を現在も持っている。
- もう片方は事故現場から教団に持ち去られた。
- 2つは、親友同士のつながりを示す重要な証拠になる。
- 2つが近づくと、震える、冷たくなる、音が鳴る、光るなどの異常反応を起こす可能性がある。
- 終盤で、NPCが自分のキーホルダーを握ったまま、盗まれた片割れを儀式の炎へ返す展開が有力。

### 誕生日プレゼントの包み

- 誕生日旅行で渡されるはずだったプレゼント。
- 中身は、ライオンのネックレスと短い恋文。
- 主人公探索者はかっこいいライオンが好きだが、NPCはかわいいライオンが好きなため、ネックレスはなんとなくかわいい雰囲気のデザインになっている。
- ライオンのネックレスには、「ずっと一緒にいたい」「あなたを独占したい」「守りたい」という深い愛情と絆の意味を込める。
- 短い恋文は、NPCが親友へ渡せなかった告白として扱う。
- 未開封のまま事故現場に残された。
- 誰から誰へのプレゼントかは、序盤では明かしすぎない。
- 教団はこれを「渡されなかった祝福」「叶わなかった未来の器」として儀式に使った。
- 開けるか、開けずに返すかは終盤の選択肢にできる。
- どちらの場合でも、目的は思い出の否定ではなく、教団が勝手に縛った未練をほどくことにする。

## 儀式と帰還条件

元の世界へ戻るには、教団が儀式に使った遺留品を取り戻し、未練を断ち切るように儀式の炎で燃やして消し去る。

ただし、燃やす意味は「思い出を消す」ことではない。

> 教団が勝手に縛り直した未練を、送り火として焼きほどく。

儀式の炎を起こす場所:

- 教団拠点内の、椅子がぽつんと一つだけ置かれた一室。
- 部屋には炉がある。
- 椅子は、呼び戻される者を迎える席であり、同時に召喚儀式の焦点として扱う。
- 儀式では炉に火を入れ、遺品を投げ入れる。
- 実際の儀式構造はこの部屋一つだけで完結せず、炉の部屋の前後左右に配置された部屋の物品が関係している。
- 中心の炉部屋は儀式の焦点であり、前後左右の部屋は境界を支える補助点として扱う。
- 前後左右の部屋に置かれた物品を調べる、動かす、回収する、壊す、または正しい位置へ戻すことで、儀式の成立条件や帰還条件に影響を与える。
- 前後左右の部屋には、持ち帰れる可能性のあるアーティファクトを配置する。
- アーティファクトは、帰還後にNPCとの記憶、関係値、次シナリオへの引き継ぎ、または特殊選択肢に影響する報酬候補として扱う。
- ただし、すべてを安全に持ち帰れるとは限らず、持ち帰りには信頼度、儀式の安定度、選択肢、または代償を絡める。
- 部屋は祭壇然としすぎず、むしろ日常から切り離された空白のような印象にする。

帰還条件の方向性:

- 教団が持ち去った遺留品を取り戻す。
- NPCが親友の死と、自分の未練を言葉にする。
- 探索者が「自分は死んだ親友の代わりではない」と宣言する。
- おそろいのキーホルダーの片割れ、誕生日プレゼントの包みなどを儀式の炎へ返す。
- 主人公探索者とNPCが殺めてしまった親族二人の遺品、親族夫婦の結婚指輪を使い、教団儀式の構造を再現する。
- 遺留品を燃やすことで、教団が作った境界の錨を消す。
- 境界の錨が消えることで、探索者は元の世界へ戻れる。

## 終盤の感情線

- NPCは、親友が帰ってきたのではないと認めなければならない。
- 探索者は、死んだ平行世界側の自分の代わりにはなれないと伝えなければならない。
- 教団は、救われたい願いのために他者の遺留品と未練を利用したことを突きつけられる。
- 主人公探索者とNPCは、互いに相手を守るために人を殺めてしまった事実を抱える。
- 二人は、被害者であると同時に加害の記憶を共有した共犯として、帰還か残留かを選ばなければならない。
- 遺留品を燃やす場面は破壊ではなく、返還と送り火として描く。

想定台詞の方向性:

```text
忘れるんやない。
返すんや。
あの子のものを、あの子のところへ。
```

```text
救われたいからって、死んだ人の声まで使わんといて。
```

## 今後詰める要素

- NPCを元の世界へ連れて帰れるか、残すかの分岐条件。
- 戦闘または対立が発生する場合の勝利条件。
- 前後左右の部屋に置く、持ち帰れる可能性のあるアーティファクトの具体案。
